# 1. 文件目的

本文件用於降低 AI 生成台詞時常見的用詞問題。

主要處理：

- AI 高頻抽象詞
- 主題宣言詞
- 成長總結詞
- 通用奇幻 / 科幻套語
- 告別、和解、犧牲場景套語
- 過度現代心理詞
- 不符合世界觀的詞
- 不符合角色聲線的詞
- 不符合劇情階段的詞
- 會提前劇透的詞
- 已在文本中過度使用的詞

本文件不是單純的「不能用詞表」。  
它同時定義：

- 哪些詞全作品禁止
- 哪些詞只在特定場景禁止
- 哪些詞只對特定角色禁止
- 哪些詞前期禁止、後期解禁
- 哪些詞不是禁止，但需要降低頻率
- 若出現禁用詞，應如何修正
- 如何避免只做同義詞替換

本文件應搭配：

- `00_protocol/00_a_台詞生產協議.md`
- `00_protocol/00_b_反ai味檢查表.md`
- `01_world/01_b_世界語言規格.md`
- `03_characters/main/*_聲線卡.md`

# 2. 基本原則

## 2.1 禁用詞不是永久審美禁令

某些詞之所以禁用，不是因為它們永遠不能出現，而是因為 AI 容易把它們用成：

- 主題宣言
- 情緒標籤
- 劇情摘要
- 角色成長總結
- 通用類型劇套語
- 沒有角色聲線的漂亮句

若某詞被禁用，代表預設不能使用。  
若要使用，必須滿足明確條件，例如：

- 任務包允許
- 角色聲線卡允許
- 劇情階段已解禁
- 該詞在本世界有特殊定義
- 該詞出現會造成戲劇後果
- 使用者明確指定保留

---

## 2.2 禁用詞檢查不得只做同義詞替換

錯誤修法：

| 禁用詞 | 錯誤替換 |
|---|---|
| 守護 | 保護 |
| 命運 | 宿命 |
| 羈絆 | 連結 |
| 希望 | 光芒 |
| 黑暗 | 陰影 |
| 救贖 | 拯救 |

這類替換通常只是讓 AI 味換一個皮，沒有真正改善台詞。

正確修法應優先改變：

- 行動
- 物件
- 稱呼
- 站位
- 沉默
- 反話
- 轉移話題
- 不成熟反應
- 場景任務
- 角色遮掩方式
- 對細節的異常執著

---

## 2.3 禁用詞需分層管理

本文件將詞彙分為：

1. 全作品禁用詞
2. 全作品慎用詞
3. 世界觀不相容詞
4. 現代心理詞
5. 高風險句型詞
6. 類型劇套語
7. 角色禁用詞
8. 場景禁用詞
9. 章節階段禁用詞
10. 已過度使用詞
11. 可解禁詞

不同類型處理方式不同，不可混為一談。

# 3. 全作品禁用詞

以下詞預設不得出現在正式角色台詞中。  
若需使用，必須由任務包或使用者明確允許。

## 3.1 抽象主題詞

| 詞彙 | 禁用原因 | 可替代方向 |
|---|---|---|
| 命運 | 容易變成主題宣言與類型劇套語 | 改成具體事件、不可逆選擇、角色誤判 |
| 羈絆 | 過度通用，容易直接總結關係 | 改成稱呼變化、物件交接、行動默契 |
| 守護 | 容易抽象化保護關係 | 改成站位、命令、擋路、留下、交代小事 |
| 希望 | 容易變成空泛正向主題 | 改成具體可保住的人、物、地點或約定 |
| 黑暗 | 通用奇幻意象，容易空泛 | 改成具體威脅、制度、環境、恐懼對象 |
| 光明 | 通用正向意象，容易空泛 | 改成實際出口、火把、清晨、地名、任務結果 |
| 救贖 | 容易宗教化或主題化 | 改成補償行動、失敗修復、無法彌補的代價 |
| 代價 | 高頻抽象詞，容易變成宣言 | 改成具體損失、傷口、失去的人或權力 |
| 選擇 | 容易把劇情變成主題作文 | 改成角色當下必須做的具體動作 |
| 真正 | 常用於主題總結 | 刪除或改成角色偏見、錯誤判斷 |
| 重要 | 太泛用，缺乏戲劇重量 | 指定具體人、物、地點、時間或承諾 |

---

## 3.2 成長總結詞

| 詞彙 / 表達 | 禁用原因 | 可替代方向 |
|---|---|---|
| 我終於明白 | 直接說出角色成長 | 改成角色做出與過去相反的行動 |
| 原來 | 容易變成覺悟開頭 | 改成沉默、改口、稱呼變化 |
| 我不再逃避 | 直接說出角色弧線 | 改成角色留下、承認錯誤或做出選擇 |
| 我會面對 | 抽象成長宣言 | 改成角色具體走向問題源頭 |
| 我已經不是以前的我 | 過度總結 | 改成他人察覺變化或角色下意識行動 |
| 從今以後 | 宣言感強 | 改成短命令、承諾物件、具體安排 |

---

## 3.3 關係總結詞

| 詞彙 / 表達 | 禁用原因 | 可替代方向 |
|---|---|---|
| 我一直都相信你 | 直接確認關係 | 改成把關鍵物交給對方 |
| 我一直都在乎你 | 直接說破真心 | 改成不合時宜的責備或提醒 |
| 你是我最重要的人 | 關係總結過滿 | 改成稱呼、例外對待、危急時站位 |
| 我不能失去你 | 直接說出恐懼 | 改成阻止對方同行、故意激怒對方離開 |
| 我需要你 | 直白依賴 | 改成命令、抱怨、找藉口留下對方 |
| 我們是夥伴 | 主題化團隊關係 | 改成第一次使用「我們」但不解釋 |

---

## 3.4 告別與犧牲套語

| 詞彙 / 表達 | 禁用原因 | 可替代方向 |
|---|---|---|
| 活下去 | 犧牲場景套語 | 改成具體小事、任務、路線、物件交代 |
| 你一定要活下去 | 過度直接 | 改成不讓對方回頭、不讓對方留下 |
| 對不起 | 高風險告別詞 | 若使用，必須造成明確關係後果 |
| 謝謝你 | 告別套語 | 改成笨拙道別或不完整讓步 |
| 再見 | 太直接 | 改成不說再見，或用日常句替代 |
| 保重 | 套語感強 | 改成具體提醒，例如裝備、路線、傷口 |
| 我會回來 | 易形成廉價承諾 | 改成迴避、玩笑、錯誤保證 |
| 等我 | 常見告別句 | 改成具體時間、地點或故意不給承諾 |

# 4. 全作品慎用詞

以下詞不是完全禁用，但需控制頻率與語境。

## 4.1 慎用抽象詞

| 詞彙 | 使用限制 |
|---|---|
| 家 | 不應泛化為溫情主題；需有具體地點、人或物 |
| 死 | 可用，但不同階級與信仰應有不同說法 |
| 神 | 視世界觀禁忌決定是否可直呼 |
| 王 | 注意階級稱謂與政治立場 |
| 自由 | 高風險政治 / 主題詞，需具體化 |
| 正義 | 容易變成說教，需角色偏見支撐 |
| 真相 | 容易變成劇情說明，需資訊控制 |
| 怪物 | 需確認是官方稱呼、貶稱或俗稱 |
| 人類 | 非人設定中需定義使用者與語境 |
| 罪 | 宗教 / 法律語感需區分 |

## 4.2 慎用情緒詞

| 詞彙 | 使用限制 |
|---|---|
| 害怕 | 角色是否習慣直說恐懼？ |
| 痛苦 | 是否可用行動或身體反應替代？ |
| 孤獨 | 是否太現代心理語言？ |
| 後悔 | 是否可用避開某人某地替代？ |
| 擔心 | 是否可用責備、檢查、命令替代？ |
| 憤怒 | 是否可由語氣、稱呼、句長表現？ |
| 悲傷 | 是否可由沉默、重複動作、錯位反應表現？ |
| 愛 | 角色是否有能力直說？世界是否允許？ |

# 5. 世界觀不相容詞

本節用於記錄不符合本作品時代、文化、科技、宗教或語言系統的詞。

## 5.1 現代不相容詞

| 詞彙 | 禁用 / 限制原因 | 可替代方向 |
|---|---|---|
| 壓力 | 可能過於現代心理 / 職場語 | 依世界語言改成負擔、催逼、扛不住等 |
| 創傷 | 現代心理詞 | 改成傷、舊事、那晚、病根等世界內說法 |
| 焦慮 | 現代心理詞 | 改成坐不住、手抖、睡不著等表現 |
| 溝通 | 現代管理語 | 改成談、說清楚、問明白、攤開等 |
| 關係 | 現代心理 / 社交詞 | 改成交情、名分、舊帳、牽扯等 |
| 自我 | 現代心理 /哲學詞 | 視作品世界決定替代語 |
| 情緒 | 現代心理詞 | 改成火氣、心病、臉色、脾氣等 |

實際禁用情況需依 `01_world/01_b_世界語言規格.md` 判定。

## 5.2 科技 / 魔法 / 世界系統不相容詞

| 詞彙 | 為何不相容 | 可替代方向 |
|---|---|---|
| 魔法 / 法術 / 咒語 / 神蹟 | 本作無魔法、無神（能量觀純物理＋科技） | 改用科技、火力、設備、能源詞 |
| 神啊 / 上帝 / 老天保佑（宗教感嘆） | 本作無宗教（見 01_a §12、02_b §11） | 改用世俗粗話、軍事 / 黑市黑話、厭世吐槽 |
| 靈魂 / 命運 / 天命 | 通用奇幻 / 宗教意象，與成本邏輯世界違和 | 改成具體事件、報表、合約、不可逆選擇 |

### §現代違和詞（議題 10.3 衍生 append）

本作科技基準為「核心都市高科技＋中低階二戰後工業感」，下列當代地球 / 網路時代詞屬現代違和，預設禁用：

| 詞彙 | 禁用原因 | 可替代方向 |
|---|---|---|
| 手機 / 上網 / 滑手機 / App | 當代地球品牌 / 網路時代用語 | 改用「終端」「頻道」「訊號」 |
| 演算法 / 大數據 / 雲端 | 當代資訊產業流行語 | 改用「系統」「資料」「規則型 AI」 |
| 直播 / 社群 / 按讚 | 網路時代用語 | 刪除或改成「通報」「口耳」 |
| 心理諮商 / 創傷後壓力 | 現代心理 / 臨床語（亦見 §5.1） | 改成「舊事」「病根」「那晚」 |

### §語言層級詞（議題 10.7 衍生 append）

由語言層級切片（官方語 / 民間語 / 階級語）抽出的禁用 / 慎用 / 推薦傾向：

| 詞彙 / 句型 | 規則 | 說明 |
|---|---|---|
| 史詩奇幻翻譯腔（「願你的劍永不卷刃」式） | 禁用 | 違作品語言基調（01_b §3.2） |
| 日式 RPG 熱血宣言（「為了大家我絕不放棄！」） | 禁用 | 違黑色幽默 / 厭世基調 |
| 軍事 / 合約 / 成本術語（彈藥、座標、風險等級、條款、損耗） | 推薦 | 官方語與前線語的核心質感 |
| 厭世吐槽句（把見死不救說成條款式冷嘲） | 推薦 | 主角聲線基調，但避免變成主題宣言 |

### §陣營詞（議題 10.8 衍生 append）

各陣營禁忌 / 不會使用的詞（對齊 01_c §4.1 / §6.1）：

| 陣營 / 群體 | 禁用詞 | 原因 |
|---|---|---|
| 黑翼防衛（官方 / AI 客服） | 直接認錯、給承諾、「我們對不起你」 | 組織把責任切割化，不會直認 |
| 前線傭兵 | 熱血宣言、主題句、心理直說 | 違厭世粗糙聲線 |
| 外圍居民 | 軍事 / 技術術語 | 身份不符 |
| 黑市 / 掮客 | 官方正式名、直說罪名 | 暴露身份 / 風險 |

# 6. 高風險句型觸發詞

以下詞常出現在 AI 式句型中，需搭配整句檢查。

## 6.1 轉折總結觸發詞

| 觸發詞 | 高風險句型 |
|---|---|
| 不是 | 不是 A，而是 B |
| 不只是 | 這不只是 A，更是 B |
| 真正 | 真正重要的是 |
| 其實 | 我其實一直 |
| 終於 | 我終於明白 |
| 至少 | 至少我們還有 |
| 也許 | 也許……但…… |
| 從今以後 | 從今以後，我會 |

## 6.2 檢查方式

若出現上述觸發詞，需檢查：

- 是否形成工整對稱句？
- 是否形成主題宣言？
- 是否形成角色成長總結？
- 是否角色此刻真的會這樣說？
- 是否可以改成更短、更偏、更不完整？

# 8. 場景禁用詞

不同場景需要不同禁用詞。  
尤其是告別、和解、犧牲、告白、反派理念、主角覺醒等高風險場景。

| 場景類型 | 禁用詞 / 句 | 禁用原因 | 替代方向 |

| 犧牲 | 我會守護你、這是我的選擇 | 主題宣言 | 具體阻擋、命令對方離開 |

| 主角覺醒 | 我終於明白、我不再逃避 | 成長總結 | 選擇與行動 |

# 11. 可解禁詞

有些禁用詞可在特定條件下使用。  
使用時必須記錄原因。

## 11.1 解禁條件

| 詞彙 | 可解禁條件 | 使用限制 | 使用後果 |
|---|---|---|---|
| 對不起 | 角色極罕見破防，且會造成關係轉折 | 每章限制次數 | 對方必須有反應 |
| 我們 | 關係已完成特定變化 | 第一次使用不可被解釋 | 玩家自行察覺 |
| 真名 | 身份揭露後 | 僅特定角色可用 | 會改變權力關係 |
| 神名 | 宗教禁忌被打破 | 需引發旁人反應 | 形成場景壓力 |

以上為示範格式，實際內容依專案設定修改。

# 12. 替代策略總表

當檢查到禁用詞時，優先參考以下替代策略。

| 問題類型 | 不建議 | 建議替代 |
|---|---|---|
| 抽象主題 | 希望、命運、守護 | 具體人、物、地點、站位、選擇後果 |
| 關係直說 | 我相信你、你很重要 | 物件交接、稱呼變化、例外對待 |
| 情緒直說 | 我很害怕、我很痛苦 | 句長變化、沉默、轉移、錯誤攻擊 |
| 犧牲套語 | 活下去、我會回來 | 小事交代、不回頭、故意激怒對方 |
| 成長總結 | 我終於明白 | 做出不同於過去的行動 |
| 世界觀說明 | 正式名詞連發 | 俗稱、偏見、誤稱、局部資訊 |
| 現代心理詞 | 創傷、焦慮、關係 | 世界內語言、身體反應、行為表現 |

## 13.1 Codex 行為限制

Codex 檢查禁用詞時：

1. 若任務要求只檢查，不得修改台詞。
2. 若任務要求修正，應先列出命中詞，再修改指定文件。
3. 不得擅自修改 LOCKED 文件。
4. 不得只做同義詞替換。
5. 若禁用詞出現在角色聲線樣本中，需確認是否為刻意示範錯誤台詞。
6. 若禁用詞出現在任務包「禁用詞」欄位中，不應判定為違規。
7. 若禁用詞出現在 QA 報告中，不應判定為台詞違規。
8. 檢查時應區分正式台詞、分析文字、模板欄位與引用說明。

# 14. 檢查範圍規則
