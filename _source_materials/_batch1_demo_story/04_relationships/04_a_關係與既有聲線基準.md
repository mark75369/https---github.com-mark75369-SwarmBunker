## Part 1 — 關係動態摘要

### 1.1 一句話定調
一個把所有情感都包進嫌棄、抱怨、社畜黑色幽默裡的傭兵「清道夫」，撿了一個從村子逃婚出來、極度渴望「被需要 / 有用」的少女瑟琳。表面是「老闆 vs 拖油瓶」的雇傭關係，底色卻是逐步生長出來的依賴、信任與不肯說出口的守護。

### 1.2 權力差（明顯不對等，但單向遞減）
- 結構上完全不對等：清道夫是救命恩人、戰鬥決策者、基地主人（「去我的基地就要遵守我的規矩」）；瑟琳是被救者、附屬、「嚮導」「助手」，自我定位極低（「我什麼都願意做」「不想成為你的累贅」）。
- 但權力差是**單向收窄**的：瑟琳從「連槍都拿不穩的村姑」→「換彈匣比前幾天俐落」→「動作流暢、戰鬥時手不抖」。清道夫的內心獨白也從「拖油瓶」轉為「我得重新評估這傢伙的價值了」。
- 瑟琳對權力差是**心甘情願甚至主動強化**的（A_03 主動下跪「用身體侍奉」以證明價值）——她要的是「被認定有用」，不是平等。這是這段關係最核心的不對稱與張力來源。

### 1.3 稱呼系統（聲線指紋的關鍵）
- 清道夫 → 瑟琳：「村姑」（最高頻、半貶半親暱）、「小鬼」、「妳這白痴」、「拖油瓶」（內心）、偶爾直呼「瑟琳」（**只在正經 / 下指令 / 計畫變更時**，如「計畫變更，瑟琳」「瑟琳，拿好裝備」）。稱呼越正式 = 情勢越嚴肅。
- 清道夫自稱：「社畜」「倒楣傭兵」「老子」「清道夫」「被高層推出來送死的」。
- 瑟琳 → 清道夫：**始終是「傭兵先生」/「您」**（從第一場到性愛場景幾乎不變），即使關係已極親密。這個敬語落差本身就是她「下位者自我認知」的聲線標記，不可隨意改成「你」。
- 瑟琳自稱：「我」，常用「那個……」開頭（怯懦試探）。

### 1.4 情緒底色
- **清道夫**：厭世、抱怨、嘴硬、毒舌，但底層是務實的善良。所有溫柔都用「嫌棄 + 找藉口」包裝（「加班費就不跟妳計較了，免得傳出去說我欺負難民」「就當作是多了個免費的人體抱枕吧」）。不會直說關心，只會行動（先把人扯到身後、留下陪睡）。
- **瑟琳**：明亮、討好、求生欲強、低自尊但韌性高。把危險詮釋成「刺激 / 有趣 / 第一次活著」，把對方的毒舌主動解讀成稱讚（「那我就把這當作傭兵先生的稱讚收下囉！」）。情緒易被「有用 / 被需要」觸發到落淚。

### 1.5 相處模式
- **嘴砲—接梗循環**：清道夫吐槽 → 瑟琳不退反進、自我合理化或正面接住 → 清道夫無奈收尾（「隨妳的便」「少得意忘形了」）。
- **行動勝於言語**：清道夫的關心 100% 透過行為（掩體、扯到身後、教她瞄準、妥協留下陪睡），語言永遠口是心非。
- **主僕化的親密**：瑟琳主動把關係推向「侍奉 / 報恩 / 忠誠度」框架；清道夫表面玩世不恭地收下，行為上卻一再讓步（沒推開、和衣躺下、沒踢出去）。
- **日常後勤節奏**：白天出勤戰鬥、晚上回基地保養裝備—這是兩人對話最放鬆、最像「相處」的時段（A_02 夕陽交心、A_03 夜間擦槍）。

### 1.6 什麼話題會緊張 / 放鬆
**會緊張 / 起防衛的話題：**
- 瑟琳的「過去 / 動機」被點破時 → 她會「才、才沒有！」式結巴否認（逃婚、被當財產、做惡夢）。
- 清道夫被瑟琳推向「情感 / 戀愛 / 溫存」框架時 → 立刻用嫌棄擋回去（「我可沒興致陪妳玩什麼事後溫存的戀愛遊戲」「妳該不會是想上演用身體報恩的狗血戲碼吧」）。
- 戰況異常 / 數量不對 / 計畫變更 → 清道夫切換成短促、嚴肅、直呼「瑟琳」的指令語氣，瑟琳立刻「是！」收斂。
- 瑟琳自我價值受質疑（「閉著眼睛亂掃」）→ 她會急著嘴硬反駁「我才沒有！手真的很穩了！」

**會放鬆 / 鬆動的話題：**
- 戰後清場、裝備保養、即溶咖啡 / 口糧吐槽 → 進入鬥嘴日常，雙方最自在。
- 瑟琳被（變相）稱讚「動作俐落」「沒那麼礙事了」→ 她瞬間雀躍，他口嫌體正直。
- 夜晚、獨處、燈關掉之後 → 清道夫的內心獨白會卸下武裝（「意外地並不讓人討厭」「迎來了難得平靜的夜晚」）。

## Part 2 — 聲線基準樣本（真實台詞抽樣）

### 2.1 清道夫（MainCharacter）— 10 句基準
> 指紋關鍵字：社畜 / 加班費 / 老子 / 媽的 / 嘖 / 高層白痴 / 嫌棄式溫柔 / 黑色幽默

1. 「喂！不想死就快滾過來！妳那一槍已經把蟲子全吵醒了，再待在那裡，妳跟妳那把破鐵管都會變成牠們的點心！」（L01｜初遇，毒舌救人）
2. 「一個路過的好心社畜。但妳要是再拖拖拉拉，我就當沒看見妳。」（L01｜自我定位「社畜」）
3. 「先說好，因為妳這白痴舉動害我被迫加班，等一下這筆加班費妳可得結清啊，不然我虧大了。」（L01｜「加班費」招牌梗）
4. 「看妳嚇得手腳抖成那副德性……算了，加班費就不跟妳計較了，免得傳出去說我欺負難民。」（L01｜嫌棄包裝的心軟）
5. 「嘖，一想到要踩進那種噁心的地方就覺得厭世。高層那群只會出一張嘴的白痴，到底什麼時候才肯多發點獎金給我？」（L02｜厭世＋罵高層）
6. 「計畫變更，瑟琳。」（L02｜嚴肅時才直呼其名，最短最重的語氣切換）
7. 「喂，村姑。妳剛才換彈匣的動作比前幾天俐落多了啊。」（L03｜「村姑」稱呼＋變相稱讚）
8. 「害怕？比起蟲子，我更害怕高層把我的薪水跟補給給砍了。」（A_02｜黑色幽默式自白）
9. 「先說好，別像隻八爪魚一樣纏上來妨礙我睡覺，不然明天一早就把妳踢出去。」（A_03｜口是心非的妥協）
10. 「……『不會亂動』個鬼。算了，就當作是多了個免費的人體抱枕吧。」（A_03｜獨白，武裝卸下的瞬間）

### 2.2 瑟琳（MainGirlA）— 11 句基準
> 指紋關鍵字：傭兵先生 / 您 / 那個…… / 是！ / 不想拖累你 / 有用 / 嘿嘿 / 結巴怯懦但韌性

1. 「啊？呃……謝謝你！你好，我是瑟琳……請問你是……？」（L01｜怯懦自我介紹，「那個 / 呃」起手）
2. 「我只是……不想死……也不想成為你的累贅。」（L01｜核心動機：不當累贅）
3. 「那個……傭兵先生，如果您覺得重的話，我可以再多扛兩箱彈藥！我的力氣滿大的！」（L02｜討好＋搶著有用）
4. 「啊……表現得太明顯了嗎？其實……我從來沒走過村子以外的路，這是我第一次真正出來『探險』。」（L02｜對外面世界的明亮憧憬）
5. 「是！因為我不想再拖累你，所以晚上在基地休息的時候，我都有偷偷拿空槍練習瞄準和換彈！」（L03｜默默努力，韌性）
6. 「嘿嘿，那我就把這當作是傭兵先生的稱讚收下囉！」（L03｜主動把毒舌解讀成稱讚）
7. 「但是……今天跟著您在外面四處探索，感覺……非常刺激！比在村子裡有趣太多了！」（A_02｜把危險詮釋成有趣）
8. 「我不再是誰的財產，而是一個有用的人。這種成就感……是在村子裡永遠體會不到的！」（A_02｜逃離「附屬品」的核心渴望）
9. 「才、才沒有！雖然身體很累，而且好幾次差點以為自己要死掉了……」（A_02｜被點破時的結巴否認）
10. 「但我不想只是一直被你保護……我想知道，除了搬東西和指路，我還能為你做些什麼。」（A_03｜想證明價值的決心）
11. 「我只是……自己一個人睡的時候，總是會夢到那些蟲子……如果有你在旁邊的話……我就不會害怕了……拜託你……」（A_03｜卸下逞強的依賴與脆弱）

## Part 3 — 寫作 / 配音施工備註（聲線守則）

1. 瑟琳對清道夫**永遠用「傭兵先生 / 您」**，不可改「你」——敬語落差是角色錯位自我認知的硬指紋。
2. 清道夫**只在嚴肅 / 下令 / 計畫變更時直呼「瑟琳」**；日常一律「村姑 / 小鬼」。稱呼即情緒溫度計。
3. 清道夫的關心**禁止直述**，必須走「嫌棄 + 找藉口 + 行動」三件套（虧本生意 / 免得被說欺負難民 / 免費抱枕）。
4. 瑟琳起手高頻使用「那個……」「呃……」「才、才……」（怯懦試探 / 結巴否認），收尾常用「是！」「嘿嘿」。
5. 清道夫的世界觀詞庫：社畜 / 加班費 / 老子 / 媽的 / 嘖 / 厭世 / 高層白痴 / 狗窩 / 倒楣——黑色幽默是他唯一的情緒出口。
6. 兩人最自然的對話節奏 = 戰後 / 夜間 / 後勤場景的「吐槽—接梗—無奈收尾」循環；緊張場景則壓縮為短指令 + 即時服從。
