## 逐場檢查

### S-01-01 基地清晨・派單 — PASS（輕微注意）

**瑟琳**
- 「那個……早、早安，傭兵先生！」「我、我有很小心！」結巴=字頭重複、前期密度高 → 符合 §4 前期、§8。
- 「彈匣我已經補好了！全部都壓滿了，紅色標記那排是按您上次說的順序排的。」報告具體「事」而非空話、且復用 §5 標誌語料 → 高度精準。
- 「我們村子以前換東西常常走——這次應該能讓您少繞一點」=求認可連著具體行動 → 符合 §9 檢核 5。
- 稱呼一律「傭兵先生」 → PASS（§6 硬規則）。
- 注意點（非錯）：「我力氣滿大的，搬這個箱子一點都不費力，真的」略偏可愛討好，但仍連著「沒弄錯」的能力證明，未越線到純卑微。

**清道夫**
- 「嘖。又停了一整夜。」「老子今天的民生用電額度大概就到這了」=嘴賤+社畜成本語感 → 符合 §3/§5.1。
- 「誰叫妳半夜不睡去動那堆東西的」嘴上嫌→「把它順手往自己這側挪了挪」手上收 → §3 反差句式教科書級。
- 「能用高層會留著自己賣。輪得到丟給我？」「比起白跑一趟，老子更怕薪水單上那行紅字」=成本/交差邏輯 → 符合 §5.1。
- 「多拿一份口糧」「順手把那排補好的彈匣也塞了進去」=嘴上嫌、手上多拿 → §3 模板精準。
- 無紅線（無保護誓言、無救世腔、無人格羞辱；「村姑」對事不對人格）。

**去名測試**：兩人句子互換不成立。瑟琳句去名後仍是「禮貌+結巴+往前看」，清道夫句去名後仍是「吐槽+務實動作」。**PASS**。

### S-01-02 赴中繼站的路上 — PASS

**瑟琳**
- 「我第一次知道，路也會留下以前的事」=對外面世界的亮眼句（§5）、嚮往滲出 → 精準。
- 「歪鐵柱、往下兩個彎、收購站岔口……記住了！我記東西很快的」=求認可+具體行動 → 符合 §9 檢核 5。
- 灌木叢「草倒的方向有點……算了，可能是我看錯。傭兵先生走過好多次了，一定比我清楚」=本地直覺被禮貌壓下，前期不搶話語權 → 符合中期前的關係階段，伏筆鋪得乾淨。
- 結巴密度適中、只在被嚇/緊張時出現（「對、對不起」「看、看到了」），談外面世界時不結巴 → 符合 §4 結巴規則。
- 稱呼一律「傭兵先生」 → PASS。
- 「能走到這裡，就已經很厲害了」非自憐、是明亮嚮往 → 未踩 §7 自憐紅線。

**清道夫**
- 「中繼站不會自己長腳過來等妳」「妳要再用那種眼神看一塊爛地，我們今晚就睡這塊爛地上了」=吵鬧型吐槽、不是酷哥 → 符合 §7 第 8 條反向要求。
- 「認柱子，別認方向——這鬼天氣，太陽妳看得到幾次？」=務實老手經驗判斷（30%成分）→ 精準。
- 舊路那段「那條別走……不過真要躲什麼的時候，沒人走的路反而乾淨。記著有這麼條就行」=嘴上拒絕+實務提點，反差句式+伏筆 → 符合 §3/§5.2。
- 「走到岔口時還是放慢了腳，等她跟上」=手上照顧、口上不認 → §3 精準。
- 肉磚黑色幽默「別問是什麼肉，問了妳會吃不下，吃不下我還得背妳」=公司省錢吐槽落到務實行動 → 符合 §5.1。
- 無紅線。

**去名測試**：**PASS**。互換明顯不成立。

**輕微注意（非扣分）**：「回程妳帶路……認不出來，我可不等妳」與 S-01-04「下次這種的，妳記」一致，採納判斷代替誇獎（§6 瑟琳列：用採納她判斷代替口頭誇獎）→ 跨場一致，good。

### S-01-04 回程黃昏・止損收尾 — PASS

**瑟琳**
- 「我是不是哪裡又填錯了……」=怕被當拖油瓶的自責，前期/中期過渡 → 符合 §8 被否定變形（聲音變小、急著攬錯），未滑入長篇自憐。
- 「上次聽收購站的人提過一句，就……記下來了」=報告具體來源、不邀功 → 符合 §5。
- 「好、好的！我會把這附近的路都記熟！」=被認可後求認可+具體行動承諾 → 精準。
- 「……傭兵先生，前面再過一個彎，就快到了。聲音輕但穩」=關係推進中、結巴下降 → 符合 §4 結巴遞減的成長聽覺指標。
- 稱呼一律「傭兵先生」 → PASS。

**清道夫**
- 「我整條腿差點餵蟲，牠們倒是嫌我資料不齊」「連『待審』兩個字都要排隊吐」=黑翼/報表/系統吐槽 → 符合 §3 固定吐槽對象。
- 「不關妳的事，這破系統打從一開就是設計來拖的」=撇清+替她卸責，對事不對人 → 精準。
- 「不過收購站那筆，補回來不少……算下來，今天沒虧」=成本/止損結帳邏輯 → §5.1 精準。
- 「下次這種的，妳記，省得我每趟都重算一遍」=採納她判斷代替口頭誇獎（§6）+ 反話照顧 → 精準。
- 「少得意忘形。撇開臉，語氣卻沒真在罵」=口嫌體正直 → §3。
- 無紅線（特別檢查：未出現保護誓言、未自陳善良、未浪漫表白）。

**去名測試**：**PASS**。
