# 1. 必先三件事（議題 1 / 10.1）

## 1.1 主線一句話

> 一個被高層當低價值耗材拋棄、嘴上隨時想離職的二線傭兵「清道夫」，在一片被各方算過帳、認定不值得善後的人為失控蟲災荒野裡，本想敷衍交差就收工，卻為了救人、活下去、湊齊一支孤立無援的灰鴉基地隊伍，被迫一路挖到「這根本不是天災」的真相——而真相揭穿後，世界依舊不打算為這片爛攤子負責。

## 1.2 核心衝突

> 清道夫想做的事是「用最低成本、最少加班把這趟爛差事打發掉並全身而退」；阻止他的力量是「一場被低報為普通異常查核、實則人為養殖技術外洩而失控的區域蟲災 + 一個把見死不救寫成合約條款、拒絕為低價值地區投入善後成本的制度」。他越想抽身，現場就越逼他留下來把人和爛攤子一起扛。

## 1.3 主題

> 當一個世界用成本與報表決定誰值得被善後，那些被當成耗材拋棄的人——傭兵、逃村女孩、被裁的技師、流浪研究者——能不能在「自生自滅」的制度冷漠裡，彼此成為對方不可替換的存在？（疑問式主題；非道德教訓）

# 2. 想要 vs 需要（議題 2 / 10.2）

## 2.1 主角「想要」（表層目標）

> 清道夫想要的是「把這趟低價值查核任務敷衍交差、不被卡危險津貼、不加班、全身而退回去躺平」。他用抱怨、成本計較、黑市倒賣破銅爛鐵的盤算來包裝這個目標（劇本V3 Prologue BeforeBattle_001~006；清道夫聲線卡「能不加班就不加班」）。

## 2.2 主角「需要」（內在成長 / 真正缺口）

> 他真正需要的是承認自己那條不肯說出口的底層邏輯——「能現在處理的麻煩最好現在處理」「人不該被當成本一筆勾銷」——並接受：在一個沒人會來救的世界裡，他可以選擇成為別人不可替換的依靠，而不是又一個只想把責任往下推、把人留在原地的環節。

<!-- INFERENCE: 「需要」由清道夫聲線卡「角色本質一句話：嘴賤厭世實則前線老手務實責任感，用抱怨遮掩責任感」+ 主題反推；待人類確認 -->

## 2.3 兩者如何衝突（高潮處選擇）

> 想要（抽身收工）與需要（留下扛人扛攤子）在每個關卡都對撞：每次他想走，現場就多一個值得救/值得留的人或一條挖出真相的線。高潮處（CH-08 組織棄養 + 養殖場真相揭露）他面臨的選擇是——拿到「這是人為災害、組織知情卻壓報表」的證據後，是按合約把成果一交、把人和真相一起留在原地撤離（想要），還是承認這支湊出來的隊伍已是他不肯丟下的東西、選擇長期駐守把活路走出來（需要）。劇本以 CH-09 隊伍高光 / 長期駐守落地「需要」的勝出，但代價是放棄「乾淨抽身」。

# 5. 規模定位（議題 5 / 10.5）

排序（可組合）：**c 對特定群體要命（中間級，主）＞ b 對角色要命，對世界不重要（個體級，次）**。

說明：

> 本作賭注規模是「對被困在這片外圍蟲災區的特定群體（瑟琳村落、灰鴉基地隊伍、被裁前員工、外圍居民）要命」的中間級，疊加「對主角小隊個體要命」的個體級。對整個世界/文明而言這只是「又一個低價值地區失控案例、家常便飯」（W-rules §4.1/§5.3）——這正是黑色幽默的尺度落差：對世界無關緊要，對被丟在裡面的人卻是世界末日。**明確不是 a 對世界要命的史詩級。**

# 6. 類型偏移風險清單（議題 6 / 10.6）

本作不允許走的方向（5-10 條；下游 09_f 類型偏移檢查基線）：

1. 不要把蟲災寫成「巨大跨國/跨星陰謀」——它只是一間瀕倒小加工廠買了黑市技術玩壞的爛尾（避免陰謀論化；W-rules §4.1 敘事氣質 / ORG §1）。
2. 不要把任何單一角色寫成「最終大反派」——對抗源是制度+成本邏輯+責任真空，無傳統反派（避免單一反派化；W-rules §5.3/§17.1）。
3. 不要把主角寫成救世主 / 熱血英雄——他是嘴賤厭世、用抱怨遮責任感的二線傭兵（避免救世主敘事；清道夫聲線卡偏移檢查）。
4. 不要把賭注上綱到世界級 / 文明存亡——本作是區域級事故（避免史詩化；§5 規模定位 b/c）。
5. 不要把結局寫成「組織悔改、制度修正、皆大歡喜」——真相揭穿後世界依舊不為爛攤子負責，活路是隊伍自己走出來的（避免和解結尾；W-rules §17.2）。
6. 不要把死亡 / 災害寫成煽情催淚——本作死亡常態化、流程化、黑色幽默處理（呼應 00_b §2 死亡處理偏好 / W-rules §17.1 生死觀）。
7. 不要把三女角寫成花瓶 / 純成人價值附屬——各有獨立功能與弧線（瑟琳聲線卡核心錨點；個人線屬其專屬流程）。
8. 不要在戰鬥/危急時刻讓角色長篇講解世界觀或技術原理（W-rules §13.4/§21.1 台詞禁區）。
