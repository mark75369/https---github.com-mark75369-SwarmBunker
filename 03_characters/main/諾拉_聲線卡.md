# 2. 聲線輪廓（議題 2 / UD §10.2 分析結果）

依議題 2「5 個極端場景」由 source（人設 §5.4 語氣階段 + §17 台詞範例 + 既有劇本台詞）綜合分析：

| 向度 | 判定 |
|---|---|
| 句長偏好 | 短至中，陳述句為主；解釋技術時可拉長為條列式精準長句，但不抒情、不堆疊修辭 |
| 潛台詞密度 | 高，但**藏在功利說辭裡**：嘴上說「效率／優先級」，底層其實是在乎；表面理由與真實動機刻意錯位 |
| 攻防傾向 | 防禦型為主：以「無視」「沉默」「岔開話題」迴避；攻擊時為冷靜陳述事實 + 隱含貶低（如「你們把能用的東西用得像廢鐵」），不咆哮 |
| 情緒遮掩方式 | 把情緒「研究化／數據化」翻譯掉（在乎=「生存優先級已上調」；想睡=「保持清醒是對能量的無謂浪費」） |
| 是否會自言自語 | 會，但多為**對自己的狀態回報式碎念**（「臨時據點佔用完成」「熱量補充有效」「變數消除了嗎」） |

**核心聲線指紋：**
1. 系統回報式語法：常以「偵測到…／判斷為…／修正，…／確認…」開頭。
2. 把人事物換算成「價值／優先級／效率／樣本／單元」。
3. 承認錯誤時不道歉，改以「因果關係上來說，是的」式的冷靜歸因。
4. 被威脅時不慌，回「判斷沉默可降低被丟出基地的機率」這類自我效益計算。
5. 結束對話常以「先去睡了／細節等我睡醒再談」收尾。

## §A 個性拆解

1. **表層個性**（聽得到的那一面）：目中無人、慵懶愛睡、有問不一定答、把別人基地當公園長椅佔用、對危機異常冷淡的科技怪才。
2. **內在個性**（驅動台詞的那一面）：極端偏科的價值排序機器——只要判定高價值就投入驚人專注與耐心，判定低價值就本能無視；不惡意，只是不放進高優先級。
3. **自尊來源**：知識密度與技術判斷力（「垃圾能不能變成資源，取決於使用者的知識密度」）；她的價值感建立在「能把別人眼中的廢墟挖成資源」。
4. **核心恐懼**：source 未明寫單一恐懼；最接近的是「失去探索價值的環境 / 睡眠與研究被干擾」。<!-- INFERENCE: 由價值排序邏輯推得，非源檔明文；待人類確認 -->
5. **情緒遮掩**：把情緒整個「研究化／數據化」翻譯掉——在乎說成「生存優先級已上調」，依賴說成「噪音規律、睡眠干擾低」，從不用情感詞。
6. **魅力來源**（可愛/吸引點）：口是心非的反差（用最功利的話做最在乎的事）、死魚眼半睡半醒的慵懶感、被威脅還能冷靜計算「沉默可降低被丟出去機率」的欠揍冷靜。
7. **努力與缺陷表現**：努力＝對高價值研究的高度專注與精準分工（維修機器人「分工精準得像早就排練過」）；缺陷＝對安全／可靠性／前置檢查隨便（「她只是懶得管某些會害人出事的細節」），生活能力趨近於零。
8. **壓力變形**：壓力下**更冷靜、更數據化**，而非崩潰（總綱）。細節：被威脅→自我效益計算後選擇沉默（見「偏移檢查」「聲線污染檢查」技術段）。
9. **角色差異**：易與「系統 AI / 旁白」混（見「聲線污染檢查」）；與莉娜的差別＝諾拉冷而懶、莉娜熱而暴（粗口對比）；與瑟琳差別＝諾拉零生活能力、瑟琳全生活維護。（人物層總結，細節見去名測試與聲線污染段。）
10. **不可偏移人格模板**（無論劇情怎麼推都不能變）：
    - 永遠用價值／優先級邏輯行動與說話。
    - 永遠把情感翻譯成效率語言，不直接抒情或告白。
    - 永遠保留能量管理底色（懶得管低價值事、想睡）。
    - 成長只到「願意維護高價值的人」，絕不到「變成溫柔常識社交人」。

> 由 §B 既有劇本台詞聲線基準歸納，規範既有劇本聲線如何帶進新台詞（F11+F13 / D-065）。
