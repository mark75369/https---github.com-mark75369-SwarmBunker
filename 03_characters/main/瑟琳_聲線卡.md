## 角色定位

**等級：** 主角級（a 深度模板）→ 落 `03_characters/main/`。

**一句話本質：** 外圍村落出身、天生嚮往自由的成年女性；逃出村子被男主救下，從「想證明自己有用」的新手後勤，成長為能有效提升男主槍械作戰效率、能獨立生存、並在主角返回組織時主動要求同行的「自由後勤搭檔」。

**功能定位：** 玩家最早接觸的本地人視角 + 男主身邊最貼近日常的後勤支援。她的興奮感負責表現「外面世界的吸引力」，她的新手視角負責反襯男主的老兵經驗（人設 §0 / §12.1）。

**核心錨點（不可動）：** 她不是逃婚花瓶、不是被救後依附的弱女角、不是只剩成人互動價值；她的長線是「成為能跟男主一起走得更遠的搭檔」，不是「獨立到不需要男主」（人設 §1 / §20）。

## 聲線輪廓

依 5 個極端場景（議題 2，自主跑由 agent 從人設 §5/§17 + 既有劇本 MainGirlA 綜合代答；自主授權通過）歸納：

| 維度 | 判定 |
|---|---|
| 句長 | 偏短到中等；緊張時碎句、結巴（「你、你是誰？！」「可是、可是前面——！」）；談外面世界時句子會變長、變亮。 |
| 潛台詞密度 | 低。瑟琳幾乎心口如一，把想法直接說出口（「我不想死……也不想成為你的累贅」）；潛台詞主要藏在「怕但仍往前」的行動，不藏在語言。 |
| 攻防傾向 | 防禦為主、低攻擊性；被嗆時的反擊是嘟囔 / 抗議（「哪有人這樣講話的……太過分了吧」「隨便你怎麼笑我啦！」），不是還擊或諷刺。 |
| 情緒遮掩 | 弱遮掩。會用「那個……」「對不起」「我會努力」當緩衝，但情緒（恐懼、興奮、感激）幾乎都外露。唯一會硬撐的是「怕但不認輸」（「我才沒有閉眼睛！我現在的手真的很穩了！」）。 |
| 是否自語 | 會。獨處 / 觀察時會小聲自言自語式提問（「她……真的去睡了嗎？」「這裡……怎麼會有這種房子？」）。 |

**5 題代答（議題 2 原始素材，寫入「聲線範例」）：**

1. **失去重要物品：** 「啊……對不起，是我沒收好……我、我馬上去找回來，一定還在附近！」（自責 + 立刻行動，不崩潰）
2. **被陌生人冒犯：** 「欸？！你、你是誰？！」→ 退一步、結巴，但手不會放開槍（人設 §16.1：害怕但仍抵抗）。
3. **不想做但必須做：** 「雖然知道很危險，但能幫上您的忙……我就覺得很不可思議！我去！」（用「有用」說服自己）。
4. **發現自己誤判：** 「啊……表現得太明顯了嗎？對不起，是我想得太簡單了……下次我會先確認的。」（坦承、不規避）。
5. **獨自一人自語：** 「（小聲）……外面的天空，原來是這樣的。」+ 偷偷拿空槍練瞄準（人設 §17.3 + 劇本 AfterBattle_008）。

> 5 題調性一致（怯生 + 行動力 + 明亮好奇），無 CONFLICT。

## 聲線範例

正確聲線樣本（取自既有劇本 MainGirlA，已驗證；詳見 §B）：

- 初登場：「啊？呃……謝謝你！你好，我是瑟琳……請問你是……？」（BeforeBattle/AfterBattle_006）
- 求留下：「不過……我懂一點槍械構造！我可以幫你裝填彈匣，絕對不會礙事的！」（BeforeBattle_026）
- 後勤回報：「傭兵先生，機槍彈匣已經全部重新裝填完畢！您的霰彈槍需要保養或補充彈藥嗎？」（AfterBattle_002）
- 嚮導獻策：「我小時候聽村裡的獵人說過，北邊的『第 13 號廢棄轉運站』以前是軍隊的整備中心……」（BeforeBattle_012）
- 自由宣言：「我不再是誰的財產，而是一個有用的人。這種成就感……是在村子裡永遠體會不到的！」（個人故事 017）
- 成長語氣：「嗯！我想繼續跟著你！我想看看更廣闊的世界，也想證明我可以幫上更多的忙！」（個人故事 026）

錯誤聲線樣本（不可寫）：

- ✗ 全知導師 / 解說世界觀的工整長句：「根據生態學原理，蟲群的活動週期受溫度影響……」（瑟琳的本地知識是村民傳聞口吻，不是學術解說）。
- ✗ 冷嘲熱諷攻擊型回嘴（那是男主 / 莉娜的聲線，見 §A-9 角色差異）。
- ✗ 卑微到只剩身體價值的台詞（人設 §15.5 / §20.1 明確禁止）。

## 髒話來源

議題 5 前置：作品 `00_b §2 髒話尺度 = 中至高`，但**前線傭兵 / 黑市 / 狩獵隊為高尺度**；瑟琳為**外圍村落新人**，個性怯生有禮 → 採**低尺度個人基線**。

| 子題 | 內容 |
|---|---|
| 觸發點 | 幾乎不罵髒話。受挫 / 被嗆時用「替代反應」：嘟囔抗議、結巴、自責道歉、硬撐反駁。 |
| 頻率 | 低（極限情境也只到「嘟囔」級）。 |
| 強度 | 輕。上限＝口頭抱怨（「太過分了吧……」「隨便你怎麼笑我啦！」「哼！一言為定！」）。 |
| 慣用感嘆 / 緩衝詞（3-8） | 「那個……」「欸？！」「啊！」「對不起」「呀！」「嘿嘿」「哼！」 |

> 與 V 禁用詞無衝突（瑟琳用詞全在輕度感嘆範圍）。無 CONFLICT。
> 設計意圖：瑟琳的「不罵髒話」本身就是與男主（高尺度）的聲線對照軸，需保留。

議題 6（直接構成 00_b §5 偏移基線 / 09_b 判定來源；自主跑授權通過）。

**a. 不得偏移成什麼（3-5 條）：**
1. 不得偏移成「只會哭、連槍都不敢碰」的純受害者（人設 §16.1 違反）。
2. 不得偏移成「價值只剩成人互動 / 身體奉獻」的花瓶（人設 §15.5 / §20.1）。
3. 不得偏移成「突然變強戰士 / 搶男主前線定位」（人設 §14.4）。
4. 不得偏移成「獨立到不需要男主」的脫離型獨立女角（人設 §1 / §20.1.4）。
5. 不得偏移成「全知本地導師 / 工整解說口吻」（本地知識須維持村民傳聞口語）。

**b. 應保留什麼（3-5 條）：**
1. 永遠保留「怕但仍往前」的核心（怕歸怕，還是想往前看）。
2. 永遠保留對外面世界的明亮好奇與興奮。
3. 永遠保留「想證明自己的行動能造成有效影響」的動機。
4. 永遠保留敬語 + 緩衝詞（「那個……」）+ 低攻擊性的禮貌底色。
5. 永遠保留「自由＝能自己選擇」的主題（不被村落 / 父親 / 婚約定義）。

## §A 個性拆解

1. **表層個性**（聽得到的那一面）：怯生、有禮、緊張結巴、常說「那個……」「對不起」；玩家第一耳聽到的是一個小心翼翼、怕添麻煩的村落女孩。
2. **內在個性**（驅動台詞的那一面）：強烈的行動欲、學習欲、冒險欲，以及「想確認自己的行動能對現實產生有效影響」的渴望。表層的怯，包著一個敢偷跑出村的行動者。
3. **自尊來源**：「我做的事真的能改變結果」——裝填快一秒、指對一條路、整好補給讓基地多撐一天。她的價值感建立在「有效影響」上，不在被誇獎，也不在身體奉獻。
4. **核心恐懼**：(1) 成為累贅 / 被丟回村子（「我不想成為你的累贅」）；(2) 人生重新被別人安排、自己的選擇再次不重要。
5. **情緒遮掩**：弱遮掩。她幾乎不藏情緒；唯一會硬撐的是「怕但不認輸」（「我才沒有閉眼睛！手真的很穩了！」）。緩衝靠「那個……」「對不起」而非掩飾。
6. **魅力來源**（可愛 / 吸引點）：「怕歸怕但仍往前」的反差萌 + 對外面世界發亮的眼睛 + 努力被看見時的「嘿嘿」式收下稱讚（AfterBattle_010）。
7. **努力與缺陷表現**：努力＝偷偷練瞄準 / 練換彈、主動攬活（「我力氣滿大的！」）；缺陷＝經驗不足、緊張時手抖、把話說得太滿後又心虛圓場（「那、那是因為我還沒習慣嘛！」）。
8. **壓力變形**：壓力下句子碎裂、結巴加重、道歉頻率上升，但**不會放棄行動**（總綱）。細節指向技術段「聲線輪廓（情緒遮掩）」與「偏移檢查 a-1（不得崩成純受害者）」。
9. **角色差異**：易與男主、莉娜混。差別——男主＝髒話 + 命令 + 厭世成本邏輯；莉娜＝火爆叫嚷 + 技術糾錯 + 無敬語直呼名字；瑟琳＝敬語 + 緩衝詞 + 低攻擊性 + 情緒外露。（與「去名測試」「聲線污染」呼應，人物層總結，不重複技術段細節。）
10. **不可偏移人格模板**（3-5 條人格核心，與聲線層「偏移檢查」呼應）：
    - 怕但仍往前——恐懼不會讓她停止行動。
    - 自由＝能自己選擇，不被村落 / 父親 / 婚約定義。
    - 想要「有效影響」而非單純討好。
    - 對善意敏感、知恩、會用自己的方式回應。
    - 終點是「能跟男主走更遠的搭檔」，不是脫離、也不是純依附。
