# 角色定位

> （議題 1 角色類型分類）

- **層級分類：a 主角級**（深度模板：完整聲線卡 + 偏移檢查 + 與類型氣質合規 + 弧線階段）。寫入 `03_characters/main/`。
- **一句話功能定位：** 灰鴉基地的「爛現場整合者」——玩家視角中軸，把本地嚮導、火炮技師、科技怪才、舊火炮、破設備、有限補給、錯誤情報與黑翼流程全部湊成一套勉強能活下去的現場方案的人。
- **角色本質一句話：** 這是一個表面上嘴賤厭世、隨時想離職的二線傭兵，實際上有前線老手務實責任感的人；他透過抱怨與成本計較遮掩「能現在處理的麻煩最好現在處理」這條不肯說出口的底層邏輯。
- **聲線一句話：** 他的句子隨情緒變動、整體偏中短且口語，詞彙來自軍事 / 黑翼職場 / 黑市 / 成本報表，情緒越強時越用罵與成本計較遮掩，很少直接說關心，更不會說救世理想。
- **不可替換性：** 否（不可替換）。刪除角色會失去玩家視角與三女角關係線的中軸、現場整合者功能。

# 聲線輪廓

> （議題 2 聲線測試題分析結果）

| 向度 | 判定 |
|---|---|
| 句長傾向 | 隨情緒變動：日常吐槽偏中長（一口氣抱怨完一串），戰鬥 / 危機切短句命令 |
| 潛台詞密度 | 高。關心 / 在乎幾乎不直說，藏在罵、成本計較、檢查裝備、留路線裡 |
| 攻防傾向 | 攻：冷嘲熱諷 + 用事實 / 成本壓人；防：拿規則 / 任務 / 成本擋人，轉移到小事（加班費、補給、報表） |
| 情緒遮掩 | 主要靠「抱怨 + 成本語言 + 嘴賤」三件套；嘴上推開、手上已在做（裝彈 / 留路線 / 多拿一份補給） |
| 是否自言自語 | 會。獨處時對著爛任務 / 爛公司碎念吐槽（BeforeBattle 大量內心吐槽即此聲線） |

**5 題聲線測試（agent 以源素材 + 既有劇本聲線綜合代擬；自主跑視為已授權，user 端應於 review 階段校正）：**

1. **失去重要物品：** 「……行吧，沒了就沒了。這破地方連命都不一定保得住，計較一把槍幹嘛。——但這筆帳我記下了。」（不抒情，轉成成本 / 記帳）
2. **被陌生人冒犯：** 「哦？在這種窮鄉僻壤居然還有人敢對老子大小聲。膽子不錯，可惜活不長。」（嘲諷 + 不真動氣）

## §A 個性拆解

> （SKILL-only mechanic；agent 由議題 1-8 答案綜合，自主跑視為 user 已於 Stage 3 確認）

1. **表層個性**（聽得到的那一面）：嘴賤、厭世、抱怨成性、成本斤斤計較、隨時嚷著要辭職的二線倒楣傭兵。
2. **內在個性**（驅動台詞的那一面）：前線老手的務實責任感——「能現在處理的麻煩最好現在處理，因為荒野裡沒處理的小麻煩會長成吃人的大麻煩」。這條經驗法則（非善良宣言）才真正決定他說什麼、做什麼。
3. **自尊來源**：建立在「我最懂現場、我活得下來、我能把爛攤子湊成能用方案」上，而非職位 / 戰功 / 公司認可。
4. **核心恐懼**：被困死在一個自己判斷錯、又沒退路的爛現場；以及（更深層、不會說）變成自己嘴上罵的那種「只會出一張嘴、把人當耗材」的高層。
5. **情緒遮掩**：抱怨 + 成本語言（加班費 / 津貼 / 報表）+ 嘴賤三件套；關心寫成「檢查裝備、罵對方慢、要對方站後面」；嘴上推開、手上已在做。
6. **魅力來源**（可愛 / 吸引點）：嘴很欠但真的可靠——「知道什麼時候該停、什麼時候該靠近、什麼時候把對方拉回安全範圍」；嘴推手做的反差萌。
7. **努力與缺陷表現**：努力藏在行動（出發前多拿一份補給、幫人留好路線）；缺陷是嘴太賤容易先得罪人、厭世到表面像不負責、不肯承認在乎。
