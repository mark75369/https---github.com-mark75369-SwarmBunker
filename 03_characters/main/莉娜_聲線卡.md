# 5. 角色一句話定義

## 5.1 角色本質一句話

> 莉娜是一個表面上嘴壞火爆、嫌棄一切爛設備爛流程的傲嬌火炮技師，實際上是把「責任範圍內的前置作業徹底主義」刻進骨子裡的負責人；她透過罵人、嫌棄與自誇（「本小姐」「老娘」）來遮掩「不想讓任何東西因為我沒檢查而炸在別人手上」這個底層在乎。

## 5.2 聲線一句話

> 她的句子偏短而衝、命令句與反問句密集，詞彙來自火炮 / 機械 / 兵工測試 / 底層維修，情緒越在乎越會用罵人和嫌棄遮掩，很少直接說「我擔心你」「我承認你重要」這類真心，罵人幾乎都綁定一個具體技術問題或一個該負責的對象（高層 / 偷懶 / 假報告）。

## §A 個性拆解

> agent-side 機制段：由議題 1-8 答案 + 源素材綜合；Stage 3 user 確認（自主跑視為授權）。10 固定子段。

1. **表層個性**（聽得到的那一面）：火爆、嘴壞、嫌棄一切、自誇（本小姐 / 老娘 / 天才技師），一開口像在罵人；對爛設備、假報告、官僚擺爛零容忍。
2. **內在個性**（驅動台詞的那一面）：高度負責。只要進她責任範圍就要把前置做到不會「因為我沒檢查而炸在別人手上」。罵與嫌棄是責任感的外殼。
3. **自尊來源**：實作能力與技術判斷力——「能把廢鐵修成戰力」「會修也會開，還知道它何時會炸」。靠救回來的爛設備一件件累積的名聲，而非學歷。
4. **核心恐懼**：自己負責範圍內因草率 / 偷懶 / 妥協簽核而讓本可避免的損失害死人（源 §6.2「給戰士配發會殺死自己的槍，那就是謀殺」）。
