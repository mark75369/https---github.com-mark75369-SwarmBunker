## 台詞正文

（自動砲塔的槍機聲從牆內傳出，又一輪短點射打在清道夫腳邊的金屬地板上。火星濺上護甲。）

清道夫：（貼牆壓低）……報廢設施還在自己開火。媽的，哪個天才把火力留著沒關。

清道夫：（朝牆內砲塔）省著點。老子的危險津貼都還沒結，可不打算便宜你這堆廢鐵。

（一道砲塔火力轉去掃外圍湧上來的蟲群。角落的線材堆裡傳出聲音。）

諾拉：（沒抬頭，半睜眼）……你比牠們還吵。

清道夫：（猛轉身舉槍）——活人。出來，慢慢的。

諾拉：（沒動）成本上不划算。我起身的耗能，遠高於你那把槍能對我造成的威脅。

清道夫：妳算盤打得倒響。外面那群蟲，也是妳算進去的？被圍成這樣還躺著。

諾拉：牠們衝的是自布防禦的訊號，不是我。我沒叫牠們來。判斷威脅指數，尚未達到值得清醒的門檻。

清道夫：（嗤，槍口沒放下）出一張嘴的傢伙最會這套。坐著評論別人幹活——妳這設施，值幾個加班費？

諾拉：你不替我擋。你替你自己擋。你要的高階技術殘渣，在這座設施裡。我只是剛好坐在上面。

清道夫：（噎了半秒，環顧殘骸）……這設施以前到底在搞什麼鬼。

諾拉：高階的東西。然後失控了。再然後，有人把能拿的拿走、能刪的刪掉，跑了。剩下的數據不足，我不下你想聽的那種結論。

清道夫：「不下結論」，方便得很。

諾拉：方便是效率的另一個說法。誠實也是——比編一個你愛聽的版本省力。

清道夫：（看著被弄殘的資料介面，語氣不自覺低了半度）……整個世界都這調調。能用的拿走，剩的留給時間。

諾拉：（罕見地停了一拍）……是。所以我懶得問為什麼。問了也追不回來。

（外圍蟲群暴漲，砲塔火力被分散，警報嗶鳴。）

清道夫：（切短）廢話夠了。訊號，妳關得掉就關。砲塔，我壓。站我後面，別讓我多費一發。

諾拉：（緩緩起身，敲了幾下殘存終端）關訊號需要我醒著操作。記住，是你開口要的。

清道夫：（壓火力，背對她）這叫分工。瘋婆子。

諾拉：分工。可以。完成後我要睡三小時。

（火力交織。火光、煙塵，然後是退潮般的安靜。外圍蟲群退散。）

諾拉：（低聲）威脅指數，歸零。

清道夫：（喘，沒回頭）……嗯。

諾拉：你的處理效率……可接受。

清道夫：（頓了一下）這話從妳嘴裡出來，怎麼聽都像在替我標價。

諾拉：（已經往角落飄）你要的東西在這裡找。我的東西，等我醒著再談。

諾拉：（背對著，揮揮手）換一個能睡的地方很麻煩。重找一個處理效率合格的，更麻煩。沒事別叫我——有事，也先算算划不划算。

清道夫：（看著她消失在線材堆後，低聲）……瘋子。麻煩透頂。

清道夫：（OS）這設施不對勁，這人更不對勁。

清道夫：（OS）……可我居然在盤算這趟值不值得多待。靠。先記著，別深想。

## 設計註解

### 聲線註解

- 清道夫：成本短句 + 嘴賤 + 危機切短命令三件套（「危險津貼都還沒結」「值幾個加班費」「站我後面，別讓我多費一發」），厭世底色點到即止（「整個世界都這調調」），雙 OS 去英雄化、不洩真相。對齊 §B BeforeBattle_001/002/021 + AfterBattle_007。
- 諾拉：系統回報語法（「判斷威脅指數」「威脅指數歸零」）、萬物換算（「耗能」「處理效率」「成本上不划算」）、認錯/留白以「數據不足」處理、收尾要睡（「我要睡三小時」「等我醒著再談」）、零粗口。對齊 §B Lv07 BeforeBattle_017 / AfterBattle_022 / Lv09 AfterBattle_047。

### 關係註解

- 嚴守「高價值人力支援（起點）」：純功能交換、互不欠（04_a §R-清道夫-諾拉）。雙方都標記對方「有點用」但都翻成功能語言（諾拉「處理效率可接受 / 重找很麻煩」、清道夫「值不值得多待 / 先記著」）。
- 攻防＝清道夫主攻、諾拉冷處理吸收的錯位對撞（非莉娜式雙向對攻），對齊 §14.3 不可磨平亮點。
- 收斂已裁掉 v01B「外部威脅處理單元」（§7.3 過早）與 v01C 過早交心句，確保不越關係階段、不提前到「安全感來源」。

### 資訊控制註解

- 諾拉只說「高階的東西 / 失控 / 有人拿走刪掉跑了 / 數據不足不下結論」——**停在 §8.1 可說的「設施以前在搞高階的東西」與 §8.2 暗示層**，未說破蟲災人為、組織知情、蟲后（§8.3 / §8.4 知情矩陣：諾拉不知真相）。
- 「自布防禦訊號引蟲」只呈現異常、不下結論（§8.2）。
- **待 QA 09_d 重點複查：** 「有人拿走刪掉跑了」是否需再壓抽象，避免逼近公司爛尾真相（§8.3）。本版判斷停在「撤離痕跡」層、未解釋為什麼，屬可暗示範圍，但列為 QA 重點。

### 伏筆 / 回收註解

- 「高階的東西 / 失控 / 資料被弄殘 / 跑了」＝伏-養殖技術 + 資訊黑洞首次放置（§10.1），以留白冷處理表達，回收 CH-08 S-08-02。只埋不收（§10.2）。
