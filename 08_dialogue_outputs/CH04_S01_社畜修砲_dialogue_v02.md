## 本場功能摘要

本場一句話：兩個被高層坑的社畜邊修爛砲邊共罵高層，從互嗆同事認出彼此是同類——莉娜拒簽爛槍被流放的往事在怨氣裡被帶出，清道夫沒安慰、只用成本邏輯接住，認可全藏在嫌棄、反話與一聲代號裡，誰都不肯說破。

本場必須完成：
1. 信任建立（同類社畜定調）：共罵高層 / 共嫌爛裝備中認出彼此是同類，靠嫌棄 / 吐槽遮掩，不直說。
2. 回收莉娜流放往事（埋於 S-03-01）：拒簽爛槍報告 → 被流放外派，以怨氣帶過，不悲情長述。
3. 推進 R-清道夫-莉娜：互嗆同事 → 同類社畜信任；結尾莉娜叫代號「清道夫」表認可。

本場不能變成：坦白互訴的和解大會 / 莉娜身世悲情回顧 / 黑翼制度控訴演講 / 技術教學旁白 / 提前說破蟲災真相。

## 台詞正文

（維修區。光從通風口斜下來，落在一地油污工具與廢鐵零件上。莉娜半個身子埋在一門拆開的舊迫擊砲裡，扳手砸得叮噹響。我踢開腳邊一塊廢鐵坐下。）

莉娜：（一扳手砸在砲身上）破銅爛鐵。黑翼這群混帳，是把垃圾場清給老娘了嗎？

清道夫：別這麼說，人家很大方的——危險津貼一毛沒有，彈藥倒是堆得像不要錢。
清道夫：發槍捨不得，發炮灰大方。

莉娜：（抽出一塊燒穿的線路板，往他腳邊一扔）看清楚。火控晶片，量產貨，撐不到三輪。
莉娜：報告上寫能撐十輪。寫報告的人，有拿它連開過十輪嗎？！

清道夫：（撿起來瞄一眼，丟回去）情報部那群坐辦公室吹冷氣的飯桶，連扳機長哪樣都不知道。
清道夫：他們開的是報表，不是砲。

莉娜：（冷笑）呵。報表。老娘最懂報表。

清道夫：（瞥她一眼。她今天的火氣不全在嘴上，在手上。）
清道夫：……懂報表的天才技師。妳這火氣，不像只是被騙來這吧，瘋婆子？

莉娜：（拧螺絲的手頓了半秒，又用力起來）……騙來的算好的。
莉娜：老娘以前在中央兵工，首席測試官。改一顆螺絲要寫五頁評估，還得看那死胖子主管的臉色。

清道夫：（沒催，等著。）

莉娜：（扳手往工具箱一摔）一批步槍，連開十發就裂膛。報告要老娘簽字背書。
莉娜：老娘當場把那份狗屎甩在死胖子臉上——給戰士配發會殺死自己的槍，那不是裝備，那是謀殺！

（短暫沉默。通風口的光在浮塵裡晃。我沒接安慰，只是把手裡的槍放下。）

清道夫：（過了一會兒，低聲）……所以被打包丟來這。鳥不生蛋，死了也沒人填表的地方。

莉娜：（重新埋回砲裡，聲音悶下去）對。報表上一個被劃掉的數字。
莉娜：（自嘲地輕笑一聲）……被丟掉的破銅爛鐵，跟這堆一樣。

清道夫：（撿起腳邊一截報廢炮管掂了掂）破銅爛鐵。
清道夫：這玩意兒在妳手裡，下禮拜就能拿來轟蟲子。
清道夫：劃掉的數字，到了這鬼地方，反而是唯一還能用的那個。

莉娜：（手停了一下，沒回頭，嗓子悶悶的）……你少往老娘臉上貼金。

清道夫：（聳肩）我算成本的。沒用的東西，我懶得誇。

莉娜：（低頭擰螺絲，半晌沒說話。再開口時嗓子已經恢復了硬）……哼。勉強算你有點眼力。

清道夫：彼此彼此。妳罵的點，沒一個是空的。
清道夫：這破基地裡，總算有個人罵高層罵得比我還準。

莉娜：（敲緊最後一顆螺絲，退開半身，斜眼瞄他）……那是當然。老娘罵的，可都有憑有據。
莉娜：（別開臉）……這裡頭，就你聽得懂老娘在罵什麼。難得。

清道夫：（站起身拍灰，往門口走）同行。被同一批人坑的同行。
清道夫：（停在門口，背對著她）……砲修好了通知一聲。喂——莉娜。
清道夫：下次老子報的點，妳給我接住。

莉娜：（對著他的背影，輕輕的）……喊老娘清道夫啊，蠢傭兵。
莉娜：（低頭，嘴角壓不住）接住就接住。下次標記敢飄一公分，老娘第一發就招呼你後腦。

清道夫：（頓了半秒，擺擺手）……記帳上了。這也算加班。

## 設計註解

### 聲線註解

- 莉娜：全程自稱「老娘」；罵綁具體技術 / 對象（火控晶片量產貨 / 連開十發裂膛 / 死胖子主管）；招牌反問後綴；核心價值硬話 §B-7 原樣落地；失落用「被劃掉的數字」意象 + 自嘲撤回遮掩（不哭訴、不總結）；結尾代號「清道夫」表認可。
- 清道夫：成本黑色幽默 + 官方文書腔反諷（「他們開的是報表，不是砲」「情報部吹冷氣的飯桶」對齊 §B-2/§B-8）；探底化用 §B-10 但事件換本場；接住失落用成本邏輯（「我算成本的，沒用的東西我懶得誇」），不安慰、不直說依賴 / 保護（守 §R §5 第1條 + §11.3 去英雄化禁區）；正經處直呼「莉娜」。

### 關係註解

- 互嗆同事 → 同類社畜信任建立（§R §1）；攻防勢均力敵（§R §2），火炮 / 火控莉娜全權、探底 / 共罵清道夫推，互不搶專業（§R §6 第3條）。
- 收斂取捨：以 v01B 對攻骨架為主軸（張力 > v01A），嫁接 v01C 高潮的情緒重量（「被劃掉的數字」+ 成本接住），收尾融 v01A 克制節制與 v01B 互嗆，三版亮點皆入。互嗆底色 + 代號認可 + 不說破三底線全程守住（§R §6）。

### 資訊控制註解

- 回收 S-03-01 流放伏筆：綁拒簽爛槍具體事件，怨氣帶過，不複述戰役、不悲情總結（§15.2 全項守住）。
- 共罵停在「省成本 / 爛裝備 / 假報告 / 死胖子主管」層，不歸因蟲災成因（CH-08 禁區 §13.3）；本場未觸蟲群異常。
- 關係不命名戀愛 / 親密（§R §5 第5條 / §13.3）；認可僅反話 + 代號暗示。

### 伏筆 / 回收註解

- 莉娜流放往事：正式回收（埋 S-03-01「比報告上還像墳」→ 本場拒簽爛槍 + 核心價值落地）。
- 蟲群異常 / 人為真相：本場不觸、不歸因（CH-08 才解禁）。
