## 本場功能摘要

本場一句話：清道夫闖進一座還在自動開火的廢棄實驗室、撞見一個被蟲群圍著卻只嫌牠們吵的研究員，兩套「口是心非功能語言」首次對撞，從敵意走到暫時休戰。

本場必須完成：
1. 引入諾拉，以「冷漠登場 / 把蟲叫實驗材料 / 睡眠優先」確立數據式功利聲線。
2. 邏輯碰撞初遇：成本厭世 × 功利冷漠首次對撞 → 暫時休戰（非信任）。
3. 埋線索：廢棄實驗室＝資訊黑洞 / 高階技術殘渣；埋「實驗室殘留」伏筆（回收 CH-08）。

本場不能變成：
1. 世界觀說明會 / 任務簡報。
2. 兩人太快互相理解 / 和解。
3. 真相演講（蟲災人為、蟲后、組織責任前期全禁）。

> 本版方向（v01C）：情緒更重、但避免直說。把「兩人都在這爛世界活得很累」的厭世底色放大——但全部藏進成本 / 效率 / 累 / 睏的功能語言裡，絕不直說孤獨 / 在乎 / 疲憊。情緒靠落差與留白漏出，不靠抒情。

## 台詞正文

（砲塔火力在牆內外來回掃，蟲群的嘶鳴壓過機械聲。清道夫靠著殘壁喘了口氣，像是已經跑了一整天。）

清道夫：（低聲，半是對自己）……又一個。又一個沒人要管、自己還在那邊燒彈藥的鬼地方。

清道夫：（朝砲塔比個中指）燒吧。反正這世界的東西，最會的就是空轉到自己死。

諾拉：（角落，眼睛幾乎沒睜開）……你也是來空轉的嗎。

清道夫：（一驚舉槍，又緩緩放下半寸）……被蟲圍成這樣，妳還有力氣酸人。

諾拉：力氣是有限資源。我只把它花在值得的事上。比如，判斷你會不會開那一槍。

清道夫：那妳判斷出來了？

諾拉：判斷為，不會。你累了。累的人不浪費子彈在不構成威脅的目標上。

清道夫：（沉默一拍，收槍）……妳這雙死魚眼倒是看得挺準。

諾拉：看得準是省力的前提。看不準的人，會在這種地方死得很快。

（外圍蟲群湧動，砲塔火力分散，光影在兩人臉上晃。）

清道夫：（環顧殘骸，語氣不自覺低下來）這設施……以前是有人很認真在這裡幹過活的。現在連資料都剩不下幾行。

諾拉：被弄殘的。有人來過，拿走能拿的，刪掉不想留的，剩下的留給時間。

清道夫：（嗤，但沒什麼力氣）……熟悉。整個世界都這調調。

諾拉：（罕見地停頓）……是。所以我懶得問為什麼。問了也追不回來。

清道夫：（看了她一眼）妳這人，怪歸怪。不像那種會在這裡待著等死的。

諾拉：我不是等死。我是還沒找到比這裡更值得醒著的地方。

（兩人之間短暫的、不舒服的安靜。蟲群再次壓上。）

清道夫：（切短，像要把剛才那點什麼甩開）……站後面。訊號關掉，砲塔我頂。別讓我多費一發。

諾拉：（緩緩起身，敲終端）……可以。

諾拉：（操作著，輕聲）你頂得住的話。

清道夫：（壓火力，背對她）老子要是頂不住，妳這設施連個收屍的都沒有。所以——閉嘴，操作。

（一段交火。火光、煙、然後是退潮般的安靜。）

諾拉：（低聲）威脅指數歸零。

清道夫：（喘，沒回頭）……嗯。

諾拉：你的處理效率……比預期高。

清道夫：（頓了一下）這話從妳嘴裡出來，怎麼聽都像在算我還能用幾次。

諾拉：（已經往角落飄）你想多了。我只是不想，太快又回到一個人吵到睡不著的狀態。

清道夫：（背對著，沒接話，過了好一會兒）……記下了。

清道夫：（OS）這設施不對勁，這人更不對勁。

清道夫：（OS）……可我居然沒想立刻走。媽的。先記著，別深想。

## 設計註解

### 聲線註解

- 清道夫：厭世底色放大但全藏功能語言（「最會的就是空轉到自己死」「整個世界都這調調」），危機仍切短命令（「站後面」「閉嘴，操作」），OS 去英雄化、不洩真相。對齊 §B BeforeBattle_001 對制度怨懟 / AfterBattle_007 去英雄化。
- 諾拉：情緒更重但仍翻成數據（「力氣是有限資源」「還沒找到比這裡更值得醒著的地方」），唯一接近吐真心的一句「不想太快又回到一個人吵到睡不著」仍用「睡不著」包裝、不說孤獨。零粗口、收尾要睡。

### 關係註解

- 本版把「兩個累壞的人短暫對上眼」的情緒放到最重，但嚴守「暫時休戰、非信任、純功能」——情緒靠留白與不接話漏出，雙方都立刻用成本 / 效率撤回。
- 風險：本版最接近「太快互相理解」紅線（§3.2.2）。收斂時需確認諾拉「不想一個人吵到睡不著」是否過早交心；建議保留但壓更冷，或移到後續關係節點。

### 資訊控制註解

- 「有人來過，拿走能拿的，刪掉不想留的」逼近 §8.3（公司跑路真相）。本版停在「追不回來 / 懶得問為什麼」的厭世留白，**不說破成因**。收斂 / QA 09_d 重點檢查。

### 伏筆 / 回收註解

- 「被弄殘 / 刪掉 / 留給時間」＝伏-養殖技術 + 資訊黑洞首次放置（§10.1，回收 CH-08）。

### 待人類確認

- 諾拉「不想太快又回到一個人吵到睡不著」是否過早交心（見關係註解）。
- 清道夫是否已知姓名（本版未直呼）。
