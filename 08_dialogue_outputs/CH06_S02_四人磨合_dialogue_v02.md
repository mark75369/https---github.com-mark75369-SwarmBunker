## 本場功能摘要

本場一句話：灰鴉基地四人第一次同框、為「廢鐵要怎麼變成能用的防線」吵成一團，磨合成莉娜火炮硬體 × 諾拉資訊自動化 × 瑟琳後勤緩衝 × 清道夫現場整合的分工。

本場必須完成：
1. 隊伍成形：四人從各自湊合磨合成有明確分工的隊伍。
2. 莉娜↔諾拉技術路線互嗆（讓廢鐵變武器 vs 讓武器變聰明），火爆 × 冷處理錯位攻防。
3. 瑟琳以後勤緩衝身分打圓場，把場面從吵架拉回可協作。
4. 清道夫不下場比技術，用成本/現場邏輯拍板分工。

本場不能變成：
1. 技術說明會 / NPC 朗讀設定。
2. 四人互相理解大會 / 隊伍宣言。
3. 莉娜↔諾拉太快互相認可（只到能共事）；任何人碰蟲災真相。

> 本版（v02）收斂自 v01A/v01B/v01C：採 v01B 最尖的莉娜↔諾拉供電/火控互嗆與清道夫拍板四句、v01A 的椅子起手與四人聲線乾淨落差、v01C 收尾的遮掩式在乎（莉娜流放底色一句、諾拉「值得維護→效率可接受」、清道夫「懶得重找」）。攻防→緩衝→拍板→各自不肯認的收尾，四段節奏完整。

## 台詞正文

（主廳。半成品哨戒砲與一門半拆的迫擊砲攤滿工作台，諾拉接管的能源終端在背景低鳴。莉娜蹲在砲座旁擰螺絲，諾拉佔了全廳唯一一張完整的椅子、半瞇著眼敲終端。清道夫抱臂靠在門邊，瑟琳抱著彈藥箱跟進來。）

清道夫：（掃了一圈）……四個人，一張能用的椅子。我就知道。

瑟琳：（小聲）那個……我去搬。

清道夫：不用。讓他們先吵完，省得待會兒椅子也被拆了。

莉娜：（拍桌）誰把主電路接到我的火控線上的？！整條線都在跟那台破終端搶電！

諾拉：（沒抬頭）是我。優先級調整。火控耗能高，分配不合理。

莉娜：不合理？沒有火控，妳的「合理」拿什麼開火，拿嘴嗎？沒有我修好的砲，妳那串程式炸給誰看？

諾拉：（重複前句，慢）炸給誰看。……火控有，調度沒有，等於零。一門打不準的砲，跟一堆廢鐵的差別，只在於它更貴。

莉娜：（站起來，指她）出一張嘴的紙上談兵！妳那套程式一斷電就變智障，到時候蟲爬上牆，妳是要拿優先級表去拍死牠們嗎？！老娘把垃圾場的破銅爛鐵修成能轟穿蟲殼的砲的時候，妳還在算妳的睡眠時數！

諾拉：（淡）你修得出砲。修不出讓它打中東西的腦子。……斷電是你的供電沒接好。變數。已記錄。

莉娜：妳——！

瑟琳：（擠到兩人中間，聲音發抖但沒退）那、那個！莉娜小姐！諾拉小姐！我、我把生活區的線跟火控線分開走了！剛剛盤點補給的時候順手做的……這樣就不會搶電了，對不對？

（短暫沉默。莉娜哼了一聲坐回去。）

清道夫：（這時才走過來看那桌零件）吵完了沒。一個能修，一個能算，一個把線分好了。我看不出有什麼好吵的，除了你們都想當老大。

諾拉：我不想當老大。我想睡。

清道夫：（不理她，對全場）聽好，我只講一遍。鐵歸莉娜——沒人比她更知道哪根線會炸。數歸諾拉——沒人比她更知道哪門砲該先開。線跟補給歸瑟琳。現場誰先打、往哪退，我說了算。出事各自扛，別互相甩鍋。算下來最省。

瑟琳：（趕緊，亮）是！我會把線收乾淨，不會礙到你們！

莉娜：（嘴硬，悶聲）……要不是妳那套算法真的省了我兩成校準時間，老娘才不認。

諾拉：（盯終端）你的硬體良率，比預估高。效率可接受。

（莉娜還在瞪那門校不準的砲。半晌，悶聲。）

莉娜：……我以前在兵工廠，校一門砲要寫五頁報告，給穿西裝的死胖子簽。簽完了砲還是會炸。

（沒人接這句——這已經是莉娜能露的最多了。）

清道夫：（把話接得很輕）這裡沒人要妳寫報告。砲炸了，自己罵兩句，記下來，明天再修。

莉娜：（哼）……說得輕鬆。

清道夫：（OS）她不是在問砲。是在確認這地方會不會又把她當棄子。

莉娜：（半天，對諾拉，別過臉）……那個火控修正參數。給我。明天。

諾拉：（捧著終端，閉眼，低聲）……據點配置損耗較低。值得維護。修正參數，睡醒給。

清道夫：什麼損耗？

諾拉：沒事。我說，效率可接受。

瑟琳：（看著三人，小小聲，亮）……感覺、感覺大家會合得來呢。

清道夫：（端起咖啡，沒看她）誰跟他們合得來。我只是懶得重找一批人。

（但他沒走。瑟琳偷偷笑了一下，繼續收線。背景裡，能源終端還在低鳴。）

## 設計註解

### 聲線註解

- 清道夫：旁觀嘴賤（「除了你們都想當老大」）+ 成本/現場邏輯拍板四句（哪根線會炸/哪門砲該先開/各自扛/算下來最省），不下場比技術；用成本邏輯接莉娜的傷（「自己罵兩句，記下來」呼應其聲線範例）；嫌麻煩遮掩在乎（「懶得重找一批人」）；OS 去英雄化點潛台詞但嘴上不說。對齊 §B BeforeBattle_002 / AfterBattle_005 / AfterBattle_007 / InteractionB1_008。
- 瑟琳：被吵聲蓋過仍硬擠進兩人中間（怕但仍往前 §A-1），緩衝以「我順手把線分開了」具體後勤化解，全程敬語（莉娜小姐/諾拉小姐/傭兵先生），明亮錨小成就（「會合得來呢」是怯生觀察非熱血宣言）。對齊 §B AfterBattle_015 / AfterBattle_002。
- 莉娜：火爆拉滿——連發短句 + 長排比技術控訴 + 反問壓場 + 自稱老娘/本小姐 + 罵綁具體技術（火控線/供電/良率/校準）；流放底色只露到「寫五頁報告給死胖子簽」制度厭世層（不直說被當棄子的傷，守 §12.3 + R §5）；惱羞遮掩認可（「才不認」「給我參數」）。對齊 §B Lv05 BeforeBattle_006 / BeforeBattle_031 / B_03 BeforeSex_009 / Lv06 流放（抽語感不照抄事件）。
- 諾拉：全程零粗口、被罵到臉上仍冷處理——數據反扣（「修不出讓它打中東西的腦子」）、重複前句迴避（「炸給誰看」）、把攻擊翻成「變數。已記錄」；在乎漏出立即蓋掉（「據點配置損耗較低。值得維護」→「我說，效率可接受」）；收尾要睡。未被激到對罵（守辨識核心 §7）。對齊 §B Lv07 BeforeBattle_023 / AfterBattle_022 / InteractionC2 / R §5 主動封口。

### 關係註解

- 莉娜↔諾拉：互看不順→能共事（非信任/友誼）。錯位攻防全程保留（莉娜雙倍火力、諾拉冷處理吸收+反扣、零粗口未對罵）；認可只到嫌棄式不情願（省校準時間/良率比預估高/給我參數/睡醒給），無一句「你技術真的很厲害我服了」（守 §12.3 + §14.4）。
- 四人在乎全部遮掩：清道夫「懶得重找」、諾拉「值得維護→效率可接受」、莉娜「給我修正參數」、瑟琳「會合得來呢」；無隊伍宣言/情感和解（守 §8.2 / §14.4）。清道夫拍板＝R-* 隱性依賴加深（OS 漏半句立刻轉吐槽撤回）；瑟琳緩衝成功被默許（R-清道夫-瑟琳）。

### 資訊控制註解

- 衝突全綁「供電/火控/良率」硬體vs自動化路線（§13.1 可明說），無蟲災成因（§13.3）。
- 莉娜流放回憶停在「寫報告/死胖子簽/砲還是會炸」制度厭世層，不觸組織知情（§13.3 替代處理）。
