## 本場功能摘要

本場一句話：清道夫衝進蟲群包圍區把實驗失誤陷險的諾拉拖出來，諾拉用數據式「你目前不可替換」包裝後怕與在乎，清道夫用加班費記帳撤回漏出的關心——兩套口是心非語言對撞，「不可替換」被一次具體事件而非宣言驗證。

本場必須完成：
1. 演出救援動作（落地武力 / 現場接口維清道夫高、諾拉自保趨近零的權力差，04_a §2）。
2. 落地諾拉「不可替換期」：以數據 / 優先級語言半承認依賴，全程零抒情、口是心非（05_c CH-07）。
3. 推進 R-清道夫-諾拉 情緒債「被否認 + 加深中」：清道夫漏半句在乎、立刻用成本語言撤回（04_a §5 第 2 條）。

本場不能變成：
1. 救援煽情大會（清道夫不可說「我會保護妳 / 不能失去妳」）。
2. 諾拉被嚇哭 / 求救 / 情緒外露（違偏移基線 a.4）。
3. 兩人互相理解大會 / 把關係說滿。
4. 世界觀說明會 / 提前洩漏蟲災真相（05_d §0，本場禁觸）。

> 本版方向（v01A）：克制、短句、強潛台詞。攻防力道壓低，靠停頓、留白與功能短句堆張力；諾拉冷處理吸收為主，清道夫話少、危機切短命令，救出後用最短的記帳一句撤回在乎。

## 台詞正文

（蟲群密度異常升高。實驗設施的自布防禦失效，殘渣設備在諾拉腳邊冒火花、塌了半截通道。她被困在崩落的支架與湧上的蟲群之間。）

諾拉：（被困，冷靜報數）防線剩四分鐘。蟲群正轉向最近熱源。

（一隻幼蟲撲上工作台，被她順手用斷管撥開，動作不慌。）

諾拉：……熱源是我。

（清道夫的槍聲從通道另一頭壓進來，一輪短點射把撲近的蟲打散。）

清道夫：（貼牆推進）又是妳。說過多少次，別碰沒檢查過的東西。

諾拉：因果關係上來說，是的。

清道夫：（罵著沒停手）省下那句。手給我。

（他探身過斷掉的支架，蟲群在外圍壓上。）

諾拉：左邊那台還在放電。

清道夫：（一把扣住她手腕）我知道。閉嘴，跟緊。

（兩人壓低貼牆退出，清道夫單手壓火力、單手把她往後拽。一段純動作的撤離。蟲群被砲火與通道塌陷隔在後方。）

清道夫：（喘，鬆開手）……出來了。

諾拉：（站定，拍掉袖上灰）威脅指數，可控。

清道夫：（背過身換彈）這筆加班費妳得結清。又得收拾妳的爛攤子。

諾拉：記在帳上。

清道夫：（頓了半秒，沒回頭）……下次別。

（停頓。他像要再說，又把話換掉。）

清道夫：別碰沒檢查的東西。就這句。

諾拉：……（看著他的背）你目前不可替換。

清道夫：（停手）什麼？

諾拉：重新找一個同等耐用、高處理效率的人，很麻煩。僅此而已。

清道夫：（重新裝彈，嗤）行。那妳這顆腦袋也省著點用，別再炸了。

諾拉：先去睡了。細節等我睡醒再談。

（她搖搖晃晃往內走。清道夫看著她背影。）

清道夫：（OS）她差點就留在那裡了。

清道夫：（OS）……加班費，下次照算。先記著。

## 設計註解

### 聲線註解

- 清道夫：危機切短命令（「手給我」「閉嘴，跟緊」），罵綁具體標的（「別碰沒檢查過的東西」），救出後關心藏進記帳（「加班費妳得結清」），漏半句「下次別」立刻換成嫌棄撤回。OS 第一人稱旁白層（去英雄化、不洩真相）。對齊 §B BeforeBattle_014/021/024 + AfterBattle_005。
- 諾拉：系統回報語法（「防線剩四分鐘」「威脅指數，可控」），認錯不道歉冷靜歸因（「因果關係上來說，是的」），用最功利的話包裝依賴（「你目前不可替換」「重新找…很麻煩」），收尾要睡。零粗口、不慌、不求救。對齊 §B 第 3/9/10/11 列。

### 關係註解

- 本場 R-清道夫-諾拉 從「高價值人力支援」往「值得維護的對象」推進一格，不命名、未說破。
- 攻防＝錯位的冷處理對撞：清道夫主攻（嫌她不管安全前置）被諾拉冷處理吸收（重複前句式「記在帳上」「因果關係上來說，是的」）。對齊 04_a §R-清道夫-諾拉 攻防傾向。
- 諾拉式半破戒：說結果（不可替換）不說因（在乎），收尾要睡迴避。清道夫漏半句立刻用成本語言撤回（04_a §5 第 2 條）。

### 資訊控制註解

- 諾拉只報威脅 / 防線戰術數據，不碰蟲災成因（§13.3 / 知情矩陣 a.2）。
- 清道夫不罵組織知情真相，只停在「爛攤子」現況層（§13.3）。
- 「不可替換」以功利語言半承認，未翻譯成情感（§13.4 / §16.3）。

### 伏筆 / 回收註解

- 「實驗失誤陷險＝諾拉對安全 / 前置檢查隨便」後果回收（§15.1，埋於 S-05-01/S-06-01）。
- 「彼此不可替換」主題以救援事件 + 功利台詞驗證，不用宣言說破（§15.1）。
