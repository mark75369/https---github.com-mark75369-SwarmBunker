# 7. 場景一句話

> 本場表面上是清道夫帶瑟琳做外圍探勘、瑟琳鼓起勇氣用村落獵人知識指出冒白霧的沙地下可能有蟲洞，實際上要讓瑟琳從「怕添麻煩、只敢被動跟著」的怯生期跨進「我做的事真的能造成有效影響」的成長期、並讓清道夫從「不信任本地新人經驗」首次給出（嘴硬包裝的）認可，並避免把「被認可」寫成熱血宣言、把瑟琳的本地知識寫成學術解說、或把蟲群異常歸因到人為真相。

# 8. 場景功能

## 8.1 本場必須完成

1. **瑟琳弧線怯生期 → 成長期的啟動切點**：瑟琳第一次主動（即使遲疑、結巴）提出專業意見並被驗證為真，獲得「我的行動能造成有效影響」的成就感（05_c 瑟琳=成長期 / 被認可）。
2. **清道夫首次認可瑟琳的本地知識**：清道夫起初不信任本地人經驗，驗證為真後給出嘴硬包裝的認可（反話 / 嫌棄變體 / 綁安全的命令），不直說稱讚。
3. **推進「蟲災規模不對勁」懸念**（P / CH-02 主線功能）：本場藉白霧沙地 / 蟲洞地下結構，加深玩家「這片蟲群分布 / 地下活動不太正常」的起疑，但不歸因。
4. **埋「白霧沙地＝蟲洞」的戰術伏筆**（瑟琳獵人知識，回收於後續戰術運用）：本場首次放置該知識點，作為後續可利用的戰術線索。

## 8.2 本場不能變成

1. 不能變成熱血成長宣言（「我終於證明了自己！」/「我也能保護大家！」式）——違 00_b §1 / 09_f 類型偏移基線 + 瑟琳「明亮須錨在具體小成就、不上升史詩宣言」監看點。
2. 不能變成全知本地導師 / 工整學術解說（瑟琳的本地知識必須是「村裡獵人說過……」的傳聞口語，不是生態學原理講解）——違瑟琳偏移檢查 a-5 / 聲線範例錯誤樣本。
3. 不能變成清道夫救世主 / 聖父式照顧（照顧寫成命令 / 嫌棄 / 成本記帳「妳要是猜錯害我繞路，加班費照算」，不直說溫柔保護）。
4. 不能讓瑟琳一被認可就翻轉成自信滿滿 / 去敬語（成長是「敢提意見」一小步，不是人格突變；敬語「傭兵先生」+ 低攻擊性永不變）。
5. 不能提前說破蟲災真相 / 成因（人為養殖技術外洩）——W-rules §22.1 禁區，CH-08 才解禁；本場蟲洞 / 白霧只談「地下有蟲、危險」表象，不歸因。
6. 不能讓清道夫的認可變成「太快互相理解 / 關係跳階」（關係階段＝救命恩人 → 漸趨並肩的早期，認可只是一次嘴硬鬆口，不是並肩搭檔定型）。
