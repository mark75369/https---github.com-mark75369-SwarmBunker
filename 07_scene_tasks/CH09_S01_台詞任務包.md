# 7. 場景一句話

> 本場表面上是一場大規模蟲潮聯合防衛戰、四人各司其職把防線守住，實際上要讓玩家從「四個被制度丟下的倒楣鬼湊在一起」看見「他們已經默契到不需要說出口就能把後背交給彼此」的隊伍高光，並讓清道夫的「需要（駐守）勝過想要（乾淨抽身）」由行動落地——同時避免把這份默契寫成主題宣言、責任宣言或「我們是夥伴」式的關係說滿。

# 8. 場景功能

## 8.1 本場必須完成

1. **隊伍高光（核心）：** 透過四人各司其職的戰術協同（清道夫前線標記 / 莉娜火炮 / 諾拉自動化調度 / 瑟琳後勤路線），呈現「默契成形、彼此不可替換」的隊伍高光（06_a §22.2 S-09-01 場景目的 + 05_b §24.1 CH-09 深層功能）。
2. **「需要」勝出落地：** 清道夫弧線終點＝需要勝出、選擇駐守走出活路（P §2.3 / CH-09；05_b §24.3 清道夫 CH-09「需要勝出=選擇駐守」）。此弧線必須**由行動 / 站位 / 選擇呈現**，不可由台詞宣言（05_c §2.3 / §12.2）。
3. **回收伏筆：** 回收基地自動化 / 能源伏筆（埋於 S-06-01 / CH-06）→ 在本場防衛戰中由諾拉的自動化調度落地發揮（05_e §0 / 05_b §24.4 CH-09「回收：基地自動化(CH-06)→聯合防衛戰」）。
4. **四人協同高光各自聲線化：** 清道夫前線整合 / 莉娜火炮控場 / 諾拉自動化遠端火控與能源調度 / 瑟琳後勤路線與彈藥補給，各以自身聲線承擔自身專業，**不互搶專業**（各 R-* §6 領域分工底線）。

## 8.2 本場不能變成

1. 不能變成主題宣言 / 責任宣言大會（「我們一定要守住這裡」「這就是我們的家」式——05_b §24.5 行動式基線；清道夫偏移檢查）。
2. 不能變成告別 / 犧牲煽情場（雖為結局前場，但本作死亡 / 告別處理＝流程化、不抒情；無指定角色犧牲，禁套用犧牲套語）。
3. 不能變成關係說滿大會（「你是我最重要的人」「我們是夥伴」式——四人默契須由戰術行動 + 綁任務的關心呈現，不直說）。
4. 不能讓清道夫變救世主 / 說救世理想（去英雄化偏移基線；駐守＝成本 / 經驗法則驅動，非「拯救這片土地」）。
5. 不能讓任一角色搶他人專業（莉娜不碰諾拉的資訊 / 自動化、諾拉不碰近身火力、瑟琳不變戰士、清道夫不變全能技術角）。
6. 不能變成戰前作戰會議朗讀設定（戰術由短促功能喊話呈現，非簡報）。
