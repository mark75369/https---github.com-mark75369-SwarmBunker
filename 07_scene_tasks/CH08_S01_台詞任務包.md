# 7. 場景一句話

> 本場表面上是「基地收到一則組織例行通知」，實際上要讓四人從「還在等組織補給／撤離支援」變成「確認被徹底棄養、只能自力求生」，並避免把制度冷漠寫成煽情控訴或熱血宣言（須保持官腔程序體的冷 × 四人反應的對比所產生的黑色幽默）。

# 8. 場景功能

## 8.1 本場必須完成

1. 透過 AI 客服／任務終端的「自生自滅／不再補給」官腔通知，揭露**組織棄養**（制度冷漠；W-rules §5.3 把見死不救寫成條款；P CH-08）。
2. 呈現四人各自的反應對比（莉娜怒 / 諾拉數據化 / 瑟琳不安 / 清道夫厭世），由反應差異承載角色辨識度與黑色幽默。
3. 把情緒從「低谷（被丟下）」推進到「決定自力求生」——由清道夫整合者功能收束（不是宣言，是務實分工 / 成本盤算式的「那就自己幹」）。

## 8.2 本場不能變成

1. 不能變成煽情催淚的「被拋棄」控訴大會（risk_type 指定流程化 + 諷刺化）。
2. 不能變成熱血團結宣言（「我們一定能挺過去」式；清道夫 / 莉娜聲線卡偏移禁區）。
3. 不能讓 AI 客服／組織「直接認錯 / 給承諾 / 道歉」（02_c §陣營詞：黑翼AI客服禁直認）。
4. 不能提前說破蟲災=人為養殖技術外洩真相（鎖至 S-08-02；W-rules §22.1 / 05_d）。
5. 不能變成世界觀說明會或任務簡報（NPC／終端不得替玩家整理全部劇情）。
