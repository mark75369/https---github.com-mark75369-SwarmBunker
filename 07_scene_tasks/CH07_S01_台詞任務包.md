# 7. 場景一句話

> 本場表面上是一場吃緊的蟲潮苦戰中，莉娜在後方火炮陣地替清道夫的前線突進炸出一條路，實際上要讓兩個「互嗆了一路、誰都不肯先承認需要對方」的同類社畜，把關係從 S-03-01 的「戰技互認雛形」推到頂點——清道夫敢把整個後背交給莉娜的火力、莉娜敢喊出「你往前衝，我把後面炸乾淨，別回頭」，讓「彼此不可替換」這個 CH-07 主題由行動與託付證明，而不是靠任何一句宣言說出來。

# 8. 場景功能

## 8.1 本場必須完成

1. **火力搭檔關係抵達頂點（託付後背）**：S-03-01 是互嗆同事起點，本場是 R-清道夫-莉娜 §4 情緒債演進的「能把背後交給對方」節點——清道夫不回頭地把後方完全交給莉娜的火炮覆蓋，莉娜用「我把後面炸乾淨，別回頭」承接這份託付。
2. **驗證「彼此不可替換」主題（P / CH-07 主線功能）**：本場以「前後分工、缺一即死」的戰術現實，把 CH-07 主題「彼此不可替換」落到行動上（清道夫＝前線突進，莉娜＝後方火力覆蓋，兩者無從屬、缺一不可）。
3. **回收四人協同伏筆（埋於 CH-06 S-06-02）**：本場是 CH-06「四人分工成形 / 技術互嗆磨合」之後，莉娜清道夫這條火力線首次在真正的苦戰裡兌現協同戰果（05_b §24.4 CH-07 行「回收：四人協同」）。
4. **保留同類社畜互嗆底色**：默契抵達頂點，但兩人說話方式不變——託付仍裹在罵、綁任務的關心、反話裡（不能因為關係近了就改成溫柔並肩）。

## 8.2 本場不能變成

1. 不能變成熱血並肩宣言 / 主題宣言（「我們一起守住！」「彼此不可替換！」式直說）——違 00_b §1 / 09_f 類型偏移基線；主題必須由行動與託付呈現，不能由角色講出來（對齊 05_b §24.5 S-09-01「行動式」處理精神）。
2. 不能變成清道夫救世主 / 聖父式照顧（照顧寫成命令 / 嫌棄 / 成本記帳；「別回頭」是戰術指令 + 信任，不是溫柔保證）。
3. 不能變成莉娜技術教學旁白（火炮覆蓋的技術說明永遠帶情緒與攻擊性，邊罵邊炸，不是中立解說）。
4. 不能讓任何一方在本場直說「沒有你我活不下去 / 我信得過你」（R 段 §5 不能說出口的事；只能用反話 / 綁任務的關心 / 行動漏出）。
5. 不能把互嗆寫成真交惡，也不能把託付寫成一方碾壓另一方（R 段 §6.2 勢均力敵、雙向對攻底線）。
6. 不能提前說破蟲災真相 / 組織知情 / 蟲后（W-rules §22.1 禁區，CH-08 才解禁；戰鬥中對蟲潮密度只可吐槽「不正常」，不可歸因人為）。
7. 不能讓莉娜在前線近身搶清道夫的現場 / 路線判斷，也不能讓清道夫接管火炮 / 火控（R 段 §6.3 永不互搶專業）。
