# 7. 場景一句話

> 本場表面上是莉娜直升機空投、與清道夫並肩打火炮首戰，實際上要讓兩個「被高層坑來這鬼地方」的倒楣鬼從「初見互嗆、互看不順眼」變成「嘴上不饒人、戰技上彼此認可」的火力搭檔起點，並避免把互嗆寫成真交惡、或把戰技互認寫成熱血並肩宣言。

---

# 8. 場景功能

## 8.1 本場必須完成

1. **引入第二名隊員莉娜**：直升機空投強勢登場，立刻定調她的火爆 + 火炮技師身份（自報「老娘叫莉娜，是來支援的技師」式）。
2. **開啟火力支援線**（P / CH-03 主線功能）：本場是灰鴉基地從「只靠主角槍械防守」升級為「具備砲擊火力」的起點。
3. **互嗆 → 互認戰技的切點**：初見先互嗆（清道夫嫌空投差點砸到人 / 莉娜嫌這破地方 + 開直升機的白痴），並肩壓制蟲潮首戰後切到「嘴上嫌、實際上認可對方戰技」。
4. **埋莉娜黑翼簽字流放往事的伏筆**（回收於 S-04-01）：本場只以「被坑來這鬼地方 / 對高層的怨」帶過，不展開拒簽缺陷報告全貌。

## 8.2 本場不能變成

1. 不能變成熱血並肩宣言（「我們一起守住這裡！」式）——違 00_b §1 / 09_f 類型偏移基線 + 兩角去英雄化 / 嘴硬底線。
2. 不能變成兩人太快互相理解 / 一場戰鬥就成莫逆（關係階段＝互嗆同事起點，不是火力搭檔頂點；頂點在 S-07-01）。
3. 不能變成莉娜技術教學旁白（她的技術說明永遠帶情緒與攻擊性，邊罵邊修 / 邊罵邊炸，不是中立解說）。
4. 不能變成清道夫救世主 / 聖父式照顧（照顧寫成命令 / 嫌棄 / 成本記帳，不直說保護）。
5. 不能提前說破蟲災真相 / 組織知情（W-rules §22.1 禁區，CH-08 才解禁）。
6. 不能讓莉娜在本場就完整交代流放原因（拒簽缺陷步槍報告全貌留 S-04-01 回收）。
