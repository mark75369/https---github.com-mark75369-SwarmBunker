# 2. 場景一句話

> 本場表面上是清道夫闖進一座還在自動開火的廢棄實驗室、和一個被蟲群圍著卻只嫌牠們吵的研究員交火，實際上要讓「清道夫 ↔ 諾拉」從互不相識變成「暫時休戰、各自記下對方有點用」的邏輯碰撞初遇，並引入「高階技術殘渣線 / 廢棄實驗室=資訊黑洞」線索；**避免**讓任何一方提前說破蟲災真相、避免把諾拉寫成有求必答的服務型 AI、避免清道夫滑成救人英雄。

# 3. 場景功能

## 3.1 本場必須完成

1. **引入諾拉**：以「自動砲塔交火 → 諾拉冷漠登場（把蟲子叫『很吵的實驗材料』、睡眠優先、毫無社交）」確立她的數據式 / 功利聲線辨識點。
2. **邏輯碰撞初遇**：清道夫務實厭世 × 諾拉功利冷漠兩套「口是心非功能語言」首次對撞 → 從敵意 / 戒備走到暫時休戰。
3. **埋線索（非揭露）**：以環境 / 物件帶出「廢棄實驗室=資訊黑洞」「高階技術殘渣線」；埋「實驗室殘留（伏-養殖技術）」伏筆，回收於 CH-08。

## 3.2 本場不能變成

1. 不能變成世界觀說明會 / 任務簡報（諾拉不可朗讀蟲災成因或設施背景設定）。
2. 不能變成兩人太快互相理解 / 和解（初遇即休戰，但**不是**信任，仍是功能性算計）。
3. 不能變成反派 / 真相演講（蟲災人為性、蟲后、組織責任前期全禁，見 §13.3）。
4. 不能變成救人英雄場（清道夫不是來救諾拉的；交火是誤會 / 自衛，不是英雄救美）。
