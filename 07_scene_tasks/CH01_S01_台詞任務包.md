# 7. 場景一句話

> 本場表面上是「剛被救下的瑟琳怯生生請求留下、笨拙地學裝填彈匣保養槍械」，實際上要讓玩家從「清道夫是個被丟包的厭世二線傭兵」變成「清道夫嘴上嫌累贅、手上已默許多帶一個人」，並把瑟琳從「怕被丟回村的逃村新人」定調成「想證明自己有用的後勤起點」，避免本場最大風險——把救援寫成救世主溫情戲、或讓清道夫直說關心。

# 8. 場景功能

## 8.1 本場必須完成

1. 建立主角清道夫聲線基調（嘴賤 + 厭世 + 成本/津貼/加班費語言 + 嘴推手做），作為全作品聲線校準樣板場。
2. 引入第一個被救的人瑟琳，定調她的怯生期（敬語「傭兵先生」+ 緩衝詞「那個……」+ 怕添麻煩 + 怕但仍往前）。
3. 透過「請求留下 → 笨拙學裝填 → 清道夫嘴賤默許」完成一次微關係推進：清道夫從「想敷衍抽身」起點漏出第一絲默許收留。
4. 建立世界壓力（被高層丟包 / 通訊死角 / 低津貼 / 蟲群威脅餘波）作為厭世吐槽的具體標的。

## 8.2 本場不能變成

1. 不能變成救世主 / 溫柔聖父救援戲（清道夫偏移檢查 a-1/a-4：不接住、不治癒、不直說保護）。
2. 不能變成世界觀說明會（清道夫不可在戰後講蟲群原理 / 蟲災成因；且 CH-01 禁揭露蟲災=人為）。
3. 不能變成瑟琳卑微到只剩身體 / 討好價值的花瓶戲（瑟琳偏移檢查 a-2）。
4. 不能變成兩人太快互相理解 / 和解（關係前期一面倒、未到並肩；不可跳階）。
5. 不能讓瑟琳變成全知本地導師工整解說（瑟琳偏移 a-5：本地知識須村民傳聞口吻，且本場她還沒到獻策階段）。

常見禁止方向：不能變成主題宣言 / 角色互相理解大會 / 任務簡報 / 告別套語堆疊。
