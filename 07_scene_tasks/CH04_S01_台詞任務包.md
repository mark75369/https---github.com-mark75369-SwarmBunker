# 7. 場景一句話

> 本場表面上是莉娜邊修舊砲邊罵黑翼發爛裝備、清道夫在旁附和吐槽，實際上要讓兩人從「互嗆的黑翼同事」變成「互相認出對方是同類社畜」的火力搭檔、並回收莉娜被流放（拒簽爛槍報告）的往事伏筆，並避免把信任寫成直接的告白 / 和解、避免把流放往事說成煽情身世大悲劇。

# 8. 場景功能（欄位 8 — 本場戲劇目的）

## 8.1 本場必須完成

1. **信任建立（同類社畜定調）：** 透過共罵高層 / 共嫌爛裝備，讓清道夫與莉娜認出彼此是同一種被高層坑、只好把爛攤子修到能用的社畜；底層同病相憐浮現，但兩人都用嫌棄 / 吐槽遮掩，不直說。
2. **回收莉娜黑翼往事伏筆（埋於 S-03-01）：** 莉娜在修砲吐槽中帶出「拒簽爛槍報告 → 被流放外派」的往事；以怨氣 / 嫌棄帶過，不做完整悲情回顧。
3. **推進 R-清道夫-莉娜 關係階段：** 從 CH-03 的「互嗆 / 互認戰技」推進到 CH-04 的「同類社畜信任」；莉娜叫代號「清道夫」表認可的信號可在本場出現。

## 8.2 本場不能變成

1. 不能變成兩人坦白互訴心聲的和解大會（兩人都不直說在乎，靠互嗆 / 綁任務遮掩）。
2. 不能變成莉娜身世悲情回顧（流放往事用怨氣 / 嫌棄帶，不煽情、不長篇自述）。
3. 不能變成黑翼制度控訴演講 / 主題宣言（罵高層要綁具體爛裝備 / 具體事件，不空泛批判制度）。
4. 不能變成技術教學旁白（莉娜講設備永遠帶情緒與攻擊性，邊罵邊修，不是中立教學）。
5. 不能讓任一方提前說破蟲災真相 / 組織知情（CH-08 才解禁；見 §13.3）。
