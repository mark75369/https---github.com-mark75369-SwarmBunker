# 7. 場景一句話

> 本場表面上是四人探索廢棄養殖場、撿拾並解碼 ORG-無名加工公司的遺留文件殘片，實際上要讓玩家與四人從「這是又一場天災」的既有認知，逐步拼出「蟲災是有人買黑市養殖增量技術玩壞、爛尾跑路造成的人為災害」這條真相，並避免把真相寫成反派演講式一次性交代或四人主題宣言式感慨。

# 8. 場景功能

## 8.1 本場必須完成

1. **真相解禁（核心）：** 透過殘片解碼揭露「蟲災＝黑市養殖增量技術外洩 → 蟲后失控」這條人為災害真相（W-rules §22.1 此場為**解禁場**；05_d §0 CH-08 解禁）。
2. **回收伏筆：** 回收 S-05-01 實驗室殘留伏筆 + 前期「天災」誤導被揭穿（05_e §0 主謎團回收 + 誤導回收）。
3. **責任鏈拼出：** 諾拉以解碼旁白逐步把 ORG §2 因果鏈（產業衰退→成本壓垮→黑市詐騙→抵押貸款→養殖實驗→蟲后誕生→失控→爛尾跑路）拼到「看得見」，但揭穿後仍是「無人認領的責任真空」。
4. **四人各自震撼反應：** 莉娜怒（綁技術/責任）、諾拉數據化解碼（核心引導者）、瑟琳不安（本地記憶呼應：加工廠突然不收貨/工人消失）、清道夫厭世（成本邏輯，「果然是別人玩壞的爛攤子」）。

## 8.2 本場不能變成

1. 不能變成反派理念演講（ORG 不說話、無反派在場可演講）。
2. 不能變成世界觀說明會 / NPC 朗讀設定（真相須由殘片物件 + 諾拉解碼分段拼出，非一次性傾倒）。
3. 不能變成主題宣言 / 角色互相理解大會（四人反應須各自聲線化，不上升為「我們要為死者討回公道」式宣言）。
4. 不能讓諾拉變成全知科技神（受殘片完整度限制，斷點須標「資料不足 / 因果鏈斷點」）。
5. 不能把 ORG 擬人化成會回話 / 反擊的反派 NPC（核心不變量）。
