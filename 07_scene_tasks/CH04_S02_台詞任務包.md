# 7. 場景一句話

> 本場表面上是一次危機脫險後的撤離帶路，實際上要讓瑟琳從「被保護的後勤」變成「主動承擔帶路風險的對等搭檔」（自由搭檔期定型），並避免把這個轉折寫成熱血宣言或讓清道夫變成溫柔放手的聖父。

# 8. 場景功能

## 8.1 本場必須完成

1. **瑟琳弧線轉折：** 瑟琳主動請纓帶路，語氣由「我會努力（不拖後腿）」遞進到「讓我來 / 這次讓我帶路，你負責開槍」——自信遞進、不突兀（仍保敬語與緩衝底色）。
2. **清道夫讓位分工：** 清道夫的保護慣性與「讓位」之間的拉扯外顯——嘴上嫌 / 記帳 / 命令，行動上默許瑟琳走前面（嘴推手做）。
3. **自由搭檔期定型：** 兩人權力從一面倒漸趨並肩；瑟琳的「有效影響」由本場具體小成就（指對路 / 帶對方向）落實，不是被誇獎、不是史詩宣言。

## 8.2 本場不能變成

1. 不能變成瑟琳的熱血宣言 / 救世主覺醒（明亮感須錨在「具體帶路」這件小事，由清道夫吐槽拉回基調）。
2. 不能變成清道夫溫柔放手的聖父式「我相信妳 / 妳長大了」直說（違去英雄化偏移基線）。
3. 不能變成兩人太快互相理解、把關係說滿的和解大會（瑟琳「怕被丟回村」不可在本場被一次說破解決）。
4. 不能讓任一方提及或暗示蟲災真相成因（CH-04 禁區，見 §13.3）。
5. 不能變成瑟琳突然變強搶前線定位（她是帶路 / 後勤，不是接管火力；清道夫仍負責開槍）。

> **文風第一順位錨點（01_d W-style 未落地 → fallback）：** 以兩端聲線卡 §B 既有劇本台詞基準為文風指紋第一順位。
> - 瑟琳 §B 正例可複用語感：「那個……」「是！」起句；「傭兵先生」固定敬語；後勤回報「具體 + 主動加問」；興奮 / 自由話題句子變長變亮。本場可參考成長語氣段（§B-8「我想繼續跟著你」）但**須下調卑微、上調「想有效影響」**（§C-2 需調整後使用）。
> - 清道夫 §B 正例可複用語感：成本 / 加班費記帳遮掩在乎（「等一下這筆加班費妳可得結清」）；危機切短命令（「躲在我後面」）；嘴軟不認（「加班費就不跟妳計較了」）。

## 16.2 攻防設計
