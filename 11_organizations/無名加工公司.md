## §1 組織本質

**一句話定位：** 無名加工公司是一間外部進駐本區、技術力只有三線水準的小型蟲肉加工承包商——本作蟲災的事故源頭，現已倒閉、爛尾跑路，只以遺留文件 / 設施 / 債務形式存在。

補充：

- 它位於蟲肉產業鏈**中段**：向當地狩獵隊收購蟲屍 / 粗處理蟲肉，把甲殼與肉分離，加工成蛋白粉、壓縮肉磚等低價加工品，再賣給中盤商 / 工業材料商。不掌握狩獵端，也不掌握終端銷售渠道，只能靠加工差價生存（源檔 §6.2、§6.3）。
- 主要產品：蟲肉粉末（低價蛋白粉 / 飼料 / 軍需食品原料）、壓縮肉磚（外圍居民 / 商隊 / 低階傭兵廉價食品）、甲殼副產物（工業膠質 / 裝甲填料 / 建材 / 化工原料）、油脂與殘渣（工業用油 / 燃料添加物 / 低階化工原料）（源檔 §6.3 產品表）。
- **它不是大型邪惡企業，也不是宏大陰謀**。源檔明示其本質定位是「一間連名字都快被抹掉的小爛公司」，這正是本作黑色幽默的核心：一場區域蟲災的源頭不是知名巨企，而是利潤微薄、瀕臨倒閉的本地加工廠（源檔 §6.2、§6.1）。
- 名稱在本作中刻意不明確設定；劇情中以「原加工承包方 / 無名加工公司 / 倒閉加工公司 / 本地蟲肉加工廠 / 牧場營運方」等代稱浮動指涉（源檔 §6.2）。

## §2 對抗性質與來源

**為何是對抗源：** 它是**制度責任鏈一環的事故源頭型對抗源**，而非主動加害的反派。對抗形式是「殘留後果 + 責任真空」——它製造了災難卻把責任鏈打碎到「沒有人願意花錢拼回去」的程度（源檔 §6.1、§6.5 第四階段）。

對抗來源（因果鏈，源檔 §6.4–§6.7）：

1. **利潤壓力**：狩獵隊成本高、中盤商壓價、物流費高、加工設備維護成本高 → 公司利潤極低，遲早倒閉。
2. **黑市詐騙**：負責人透過朋友接觸黑市技術掮客，買下被包裝成「低成本蟲類養殖增量方案」的來路不明技術；為購買技術 / 改造牧場設備 / 建初期養殖測試場而**抵押貸款**。
3. **技術越界 / 實驗失控**：該技術實為粗暴刺激蟲類繁殖力、生長速度、進食需求與生物強度的危險技術（含繁殖促進劑 / 高增殖飼料 / 激素或基因刺激 / 蟲后誘導程序 / 環境壓力刺激 / 群體費洛蒙控制 / 生長週期壓縮）。公司三線技術力無法理解風險，小規模測試初期「看似成功」（產卵增 / 成長快 / 肉量高 / 成本報表改善）→ 異常出現（攻擊性升、甲殼變厚、食量暴增、隔離欄被撞壞、工人受傷、邊緣出現小蟲巢）→ 已投入大量成本＋還款壓力下不肯停 → **失控**：唯一存活母體逃入地下 / 舊設施 / 牧場深層，成為後續超巨大蟲后的源頭。
4. **棄置爛尾**：失控時公司已無力處理，還款期限逼近、武力討債公司即將上門。負責人判斷無法支付清除 / 善後 / 債務 / 員工賠償 / 非法技術責任，遂讓公司倒閉、刪除部分資料、封存部分研究紀錄、違法解雇員工、對少數知情者下封口令、對外宣稱「經營不善與區域產業衰退」。

**對抗性質定調（源檔 §6.1）：** 不寫成傳統反派。它是「一件爛事如何在沒有人真正負責的情況下，變成一場區域災害」中的**事故源頭**；黑市掮客是危險技術販售者、地方監管形同虛設、區域衰退讓災害缺乏即時處理價值——四者共同構成責任鏈，無名加工公司只是其中一環（最關鍵的源頭環）。

## §3 組織結構/層級

源檔可辨識的內部分層（源檔 §6.6 資訊不對稱結構）：

| 層級 | 構成 | 知情程度 | 是否對抗主體 |
|---|---|---|---|
| 負責人（老闆） | 公司主事者 | 知道技術來自黑市、實驗已失控、公司選擇放棄善後；主導刪資料 / 封存 / 跑路 / 封口 | **是——對抗責任的核心承擔者** |
| 極少數核心管理層 | 與負責人共同處理爛尾者 | 同負責人；參與宣告倒閉 / 違法解雇 / 銷毀封存紀錄 / 切斷牧場供電通訊 / 下封口令 | 是（次於負責人） |
| 牧場測試參與員工 | 少數參與牧場測試的人 | 知道公司曾試特殊飼料 / 繁殖方案 / 低成本養殖技術，也知道「蟲子變得不對勁」；但未必知道黑市技術來源或完整事故規模 | 否（部分受害、部分被封口） |
| 普通員工 | 多數一般員工 | 只知道經營不善 / 訂單變少 / 原料不穩 / 工廠停工 / 倒閉 / 自己被違法解雇或欠薪 | 否（受害者，被偽裝成單純經營失敗） |

**對抗主體是「負責人＋核心管理層」這一層**；普通員工是被同一組織機制坑害的受害者，可作下游 C↔ORG 同情 / 控訴向關係的素材來源。

## §4 殘留型態

**現狀：已倒閉 / 撤離 / 跑路**，以下列遺留形式被引用（源檔 §6.5 第四階段、§6.6）：

- **設施遺留**：搬走能搬走的設備，留下無法搬走的牧場與加工設施；切斷牧場供電與通訊。半廢棄加工廠 + 撤離痕跡成為前期可見線索。
- **資訊黑洞**：源檔 §3 段明示——一座已停電、斷線、被蟲群佔據的廢棄養殖場本身就是天然資訊黑洞。公司**並不需要強大情報封鎖能力**，只做了最低限度刪除 / 封存 / 跑路，剩下的資訊消失由荒廢、蟲群、通訊不良與無人關心共同完成（「不是完美滅證，而是最低限度的爛尾處理」）。
- **資料殘片（下游揭露核心；源檔 §6.6 殘片表）**：主角小隊後續查到的不是完整公司資料，而是牧場中被封存或未完全刪除的殘片——
  - 牧場本地終端紀錄（實驗數據，因通訊差未完全上傳）
  - 成本報表殘片（公司被原料成本與中盤商壓垮）
  - 黑市技術說明片段（技術被包裝成養殖方案）
  - 抵押貸款通知（資金壓力）
  - 討債警告訊息（負責人倉促撤離原因）
  - 事故日誌（蟲群失控並非自然事件）
  - 封存紀錄（管理層知道異常但選擇封起來）
  - 員工解雇通知（對普通員工偽裝成經營失敗）
- **債務 / 法律責任遺留**：抵押貸款未清、武力討債未了、違法解雇與非法技術責任未追究——以「沒人願意花錢拼回去的責任鏈」形式懸置。
