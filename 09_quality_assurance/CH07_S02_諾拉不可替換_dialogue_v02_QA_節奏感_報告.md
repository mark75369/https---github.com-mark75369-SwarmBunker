## 基線來源

- 任務包 §18.1 長度限制：總句數中（16-26 行）；戰鬥救援命令句 ≤15 字為主、日常吐槽/數據回報可中長；允許沉默、動作插入、內心獨白。
- 任務包 §16.1 節奏曲線：開場高 → 中段最高 → 轉折由高轉微妙 → 收尾反差萌。
- 09_b 聲線偏好：清道夫句長隨情緒（戰鬥切短/日常中長）；諾拉短至中陳述、技術可精準長句。

## 句長分佈（speaker 行 24，含 3 OS）

- 短命令 / 短陳述（≤10 字）：「閉嘴。跟緊，貼著我，別亂動。」「記在帳上。」「抖的是我。」「無外傷。你不必檢查第三次。」——戰鬥段與收束 punch 密集。
- 中句：「防線剩三分四十。蟲群會在崩潰後轉向最近熱源。」「因果關係上來說，是的。」
- 長句（諾拉條列式精準）：「你目前不可替換。不是修辭，我計算過。能在三分四十內到達、能單人壓制這個密度、能把我從崩塌側帶出、而且願意回來的單元——只有一個。」（最長，刻意落在收尾高光）
- 變異係數高：長短擺盪明顯（戰鬥切短、不可替換段拉長），swing 充分，非機械等長。

## 段落呼吸 / 停頓密度

- 動作旁白插入分隔對白節拍（撥幼蟲 / 連開三槍 / 反覆按肩 / 塌方封通道 / 背對著肩膀起伏），呼吸點充足。
- 省略號表停頓「……熱源是我」「……哪裡傷了」「……出來了」「下次再這樣，我就——（停）……」，停頓密度適中、皆有功能（後怕 / 漏半句）。
- 破折號「——」用於諾拉收束（單元——只有一個）與清道夫硬切（我就——），未濫用（守聲線卡負例「禁破折號濫用」）。

## 對抗式攻擊切角（先攻後判）

**Lens A（節奏指標超標即 FAIL）攻擊：**
1. 攻：戰鬥救援段命令句是否守 ≤15 字（任務包 §18.1）？是否有句子超長拖垮戰鬥節奏？
   - 攻不破：戰鬥段命令「閉嘴。跟緊，貼著我，別亂動。」「狗娘養的，妳就不能怕一次嗎？」皆切短，符合 ≤15 字戰鬥節奏。唯一長句（不可替換段）落在救出後收束段，符合「日常/數據回報可中長」例外，非戰鬥段，不拖節奏。
2. 攻：諾拉系統回報句是否單調等長、節奏死板？
   - 攻不破：諾拉句長從「記在帳上」（短）到「能在三分四十內到達…只有一個」（長條列）跨度大，由情緒強度驅動（越在乎越精確越長），節奏有起伏非單調。
3. 攻：總句數是否落在 16-26 區間？
   - 攻不破：24 speaker 行（含 3 OS），落在區間內，密度合規。

**Lens B（留白派）攻擊：**
1. 辯：諾拉「先去睡了」後的留白、清道夫破折號停頓，是否克制節奏的刻意設計？
   - 結論：是刻意設計。停頓 / 留白承載「真心邊緣各自撤退」的節奏功能（任務包 §16.2 回合 4），非節奏空洞。對齊任務包 §18.1「允許沉默」。
2. 辯：OS 三連（她連抖都沒抖 / 抖的是我 / 後面那句刪掉）節奏是否過密？
   - 結論：非過密。三 OS 短促遞進形成收尾 punch 節奏（鋪陳→反轉→自我封口），是黑色幽默落點設計。

## 節奏曲線擬合

開場（被困報數據，高）→ 中段（清道夫衝入攻防＋切短命令救援，最高密度短句）→ 轉折（出來了 / 破折號漏半句，由高轉微妙）→ 收尾（不可替換長句＋OS 反差萌）。完全對齊任務包 §16.1 曲線。
