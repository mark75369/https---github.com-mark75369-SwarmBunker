# 1. 文件目的

本文件是作品世界觀的總入口，用於整理遊戲劇本所需的世界本體資料。

本文件回答的核心問題是：

- 這是一個什麼樣的世界？
- 這個世界如何運作？
- 這個世界最重要的矛盾是什麼？
- 這個世界如何壓迫角色？
- 玩家進入世界後，應該逐步理解什麼？
- 哪些世界觀資訊會影響劇情、角色與台詞？

本文件不是百科全書。  
不需要收錄所有背景細節，只收錄會影響以下內容的世界觀資訊：

- 主線劇情
- 角色動機
- 陣營衝突
- 任務設計
- 台詞語言
- 資訊揭露
- 玩家理解
- 角色選擇
- 世界規則

若某段世界觀只是背景裝飾，且不影響角色行為、台詞、劇情或玩家決策，應放入附錄或參考資料，不必寫入本文件主體。

# 3. 世界觀一句話定義

請用一句話定義本作品世界。

此句不應是宣傳文案，而應是可供劇本判斷的設計核心。

格式建議：

> 這是一個【世界類型】的世界，其中【核心秩序】建立在【代價 / 謊言 / 資源 / 信仰 / 制度】之上，而角色必須面對【主要矛盾】。

> 這是一個高科技文明與高危荒野長期並存的世界，其中核心秩序建立在「把危險與災害外包給低價值外圍區域」的成本算計之上，而角色必須在一片已被各方算過帳、認定不值得善後的外圍蟲災區裡活下去。

# 5. 世界基調

## 5.1 玩家第一印象

玩家第一次接觸這個世界時，應該感覺到什麼？

- 視覺印象：半廢棄的灰鴉基地、撤空的據點、霧霾與污染懸浮物、零星蟲巢與舊巡邏線。
- 情緒印象：厭世、被丟包、爛攤子、低成本敷衍。
- 社會印象：高度商業化、合約化、責任切割化；高層用報表決定誰死活。
- 危險印象：蟲群數量與攻擊性遠超正常，通訊不良讓求援無門。
- 道德印象：沒有宏大反派，只有一層層便宜行事與沒人想負責。
- 神秘感來源：蟲災規模為何異常（指向黑市養殖增量實驗失控）、廢棄高階實驗室的黑科技殘渣。

一句話描述：

> 一個正常運作但高度剝削、高度資本化且高度危險的世界中，邊境人員被迫承擔核心城市與上層組織不願面對的風險。

---

## 5.2 世界的底層情緒

本作品底層情緒（從通用清單對齊源檔）：

> 主軸為**黑色幽默**與**殘酷但日常**；輔以**技術冷漠**（核心都市可從污染提煉能源，外圍卻被污染害死）與**階級窒息**（核心制定規則、外圍承受後果）。

---

## 5.3 世界最殘酷的規則

本作品最殘酷的規則（對齊源檔 §4.2 / §6.1 災害成因基調）：

> 在這個世界裡，技術不是因為太偉大才造成災難，而是因為它夠爛、夠便宜、夠危險，才會流到最沒有能力處理它的人手上；而一片地方越不重要，就越會在沒人善後的狀態下慢慢爛掉。

這條規則如何影響：

- 主角：清道夫被當成便宜耐用的多功能消耗品，派去確認一份「低價值爛差事」，結果是蟲災現場。
- 主要反派：本作無傳統反派；最接近「反派」的是無名加工公司、黑市掮客與不在乎的上層制度（詳 §17、§22）。
- 普通人：外圍居民、瑟琳村落承擔污染、災害、失敗項目與武裝衝突的後果。
- 任務設計：任務被低報為「異常查核 / 舊資產巡檢」，補給按低危等級發放，逼出塔防硬擠戰力的玩法。
- 台詞語氣：主角對高層、補給、津貼、合約條款的厭世吐槽。
- 玩家選擇：在破爛帳面資產裡修復、改造、升級灰鴉基地求生。

# 6. 世界運作規則

本節為 W-rules 主體（議題 10.2 世界規則最小集，四子題：時間觀 / 能量觀 / 神‧魔法‧科技邊界 / 社會結構基底）。

### 時間觀

線性時間。源檔無時間穿越、重設、回溯設定；故事按關卡與區域歷史線（§8.2 待 /create-outline 細化）單向推進。

### 能量觀

純物理＋科技，無魔法、無神性。力量來源是火藥武器、火炮、電池能源與（核心都市專屬的）污染能源提煉技術；行動受彈藥補給、電力、機械維修可靠性限制。

### 神 / 魔法 / 科技的邊界

- 神：不存在（源檔無神祇 / 超自然）。
- 魔法：不存在。
- 科技：存在且**高度受階級、區域、資本分割**。核心都市掌握 AI 自主設備、生物義體、高階環境淨化、污染能源提煉；外圍只能用降級版、過時版、黑市流出版。詳 §13。

### 社會結構基底

資本＋軍事承包混合。名義上有城市聯盟 / 中央行政體系，實際由大型企業、軍工機構、承包商與地方勢力共同維持運作；外圍區域規則更彈性、交易化、暴力化。

## 6.1 基本世界規則

| 編號 | 規則 | 對劇情的影響 | 對角色的影響 |
|---|---|---|---|
| 1 | 核心城市高度繁榮，外圍世界高度危險；文明未崩壞但覆蓋範圍受荒野生態、商業利益、通訊限制與武裝成本共同限制 | 確立「文明節點」式地圖；解釋為何有塔防據點 | 角色被迫在據點與荒野間求生 |
| 2 | 強大野生生物是世界常態，人類活動以城市、城鎮、中繼站、據點串聯 | 武裝護送、清理、回收成為成熟長期產業 | 持槍、修牆、巡邏、避難是日常 |
| 3 | 有權力單位不是不能處理外圍災害，而是不願為低價值地區投入高成本資源 | 本區蟲災被當低優先級查核，非正式救災 | 主角被低成本派來、被卡危險津貼 |
| 4 | 跨區 / 荒野 / 長距離通訊因污染微粒與霧霾干擾而昂貴、延遲、不穩甚至中斷 | 資訊不對稱形成套利空間；事故長期惡化不被發現 | 主角情報過時、求援困難、津貼缺證據 |
| 5 | 技術殘渣可改變低階層命運，但低階組織看不懂風險、無力善後 | 蟲災正是黑市養殖增量技術失控的結果 | 角色面對的是別人玩壞的爛攤子 |

規則應具備以下特徵：會限制角色行動、會製造代價、會造成衝突、會影響任務設計、會在台詞中留下痕跡。本作五條規則均滿足。

# 13. 魔法、科技或超自然系統

本作無魔法 / 超自然，但有**高度分層的科技系統**。本節記錄議題 10.3 科技水平（三子題：技術等級基準 / 誰擁有先進技術 / 技術階級差）。

### 技術等級基準

非平均分布的科技。整體分四層：

| 層級 | 代表勢力 / 區域 | 科技特色 |
|---|---|---|
| 最高層 | 核心都市、頂層資本、超大型組織 | AI 自主設備、生物義體、高階環境淨化、污染能源提煉 |
| 二線高階 | 主角組織黑翼防衛、大型軍事承包商、兵工體系 | 火藥武器、火炮、直升機、越野載具、野外戰鬥技術、軍事訓練 |
| 中低階 | 衛星城鎮、中繼小鎮、外圍聚落 | 二戰後至冷戰早期感的實用技術，依賴電池與部分石化燃料 |
| 特殊個體 / 殘渣科技 | 諾拉、被遺留研究設施、失敗項目殘餘設備 | 規則型 AI、自動砲塔、無人防禦系統、電池與資訊技術 |

中低階層約為「二戰後工業科技感」，但更依賴電池能源而非完全依賴石化燃料；外圍戰鬥科技以火藥槍械、炸藥、迫擊砲、火炮、重武器為主，塔防玩法建立在這一層。

### 誰擁有先進技術

最先進技術由核心都市與頂層資本掌握（不在本作正面登場，只作壓迫性背景）。主角組織黑翼防衛屬二線，弱於 AI / 資訊 / 通訊 / 高階能源，強於軍事戰鬥。平民日常只能接觸降級版、過時版、拆解版或黑市流出版。

### 技術階級差

技術階級差極大，且形成「會用 vs 不會用」「能維護 vs 不能維護」的對立。核心居民享自動化、生物義體、乾淨作物；外圍村落仍用老式槍械、溫室農業與土法改裝發電機。這不是文明倒退，而是科技被資本、環境與通訊隔閡分割的結果。

# 17. 道德結構

## 17.1 世界的道德矛盾

本節記錄議題 10.5 各項價值觀（五維度）與本作主要道德矛盾。

**五維度價值觀（對齊源檔）：**

- 生死觀（必答）：死亡常態化、流程化。對世界 / 上層而言死亡是可計價成本（「該風險等級仍在可接受成本內」）；對外圍與前線是日常風險。無來世 / 輪迴觀（無宗教）。
- 勞動觀：高度剝削。人員是「資產也是耗材」；只要任務收益高於替換成本，組織接受人員損耗。
- 家族觀：外圍村落靠血緣 / 聯姻 / 親族建立互助網以分擔防衛風險（瑟琳聯姻＝村落生存策略）。<!-- 禁忌話題見下方 §禁忌話題彙整 -->

- 自由觀：集體生存邏輯壓過個人自由（瑟琳的核心衝突即「個人自由 vs 村落生存邏輯」）。

# 23. Codex 使用規則

當 Codex 修改或建立本文件時，應遵守：

1. 不得把本文件寫成完整百科。
2. 只收錄會影響劇情、角色、任務、台詞或玩家理解的世界觀資訊。
3. 不得擅自補完重大真相。
4. 不得把 DRAFT 內容視為 LOCKED。
5. 若世界觀內容影響台詞語言，應同步提醒更新 `01B_世界語言規格.md`。
6. 若世界觀內容涉及名詞，應同步提醒更新 `02A_專有名詞表.md`。
7. 若世界觀內容涉及俗稱或黑話，應同步提醒更新 `02B_俗稱與黑話表.md`。
8. 若世界觀內容涉及禁用詞或解禁條件，應同步提醒更新 `02C_禁用詞與慎用詞表.md`。
9. 修改後必須列出影響範圍。
