# 1. 文件目的

本文件用於定義一個作品世界中的人物如何說話。

世界觀設定本身不等於台詞語言。  
一個作品可能有龐大的歷史、種族、宗教、政治、科技、魔法與事件，但角色在台詞中實際使用的語言，必須經過轉換。

本文件的目的，是把世界觀轉化為以下可執行規則：

- 這個世界的人如何稱呼事物
- 什麼詞可以說，什麼詞不能說
- 不同階級、陣營、職業、地域如何使用不同語言
- 哪些詞太現代、太抽象、太 AI 味
- 哪些世界觀名詞不應被角色直接朗讀
- 哪些概念需要俗稱、黑話、代稱或禁忌語
- 什麼程度的口語、粗口、詩性語言可以接受
- 世界觀資訊應如何透過台詞自然暴露

本文件不是世界觀百科。  
本文件只記錄「會直接影響角色說話方式」的語言規則。

完整世界觀資料應放在：

- `01A_世界觀總覽.md`
- `01C_陣營與階級語言.md`
- `02A_專有名詞表.md`
- `02B_俗稱與黑話表.md`
- `02C_禁用詞與慎用詞表.md`

# 2. 使用原則

## 2.1 語言規格高於一般文風偏好

生成台詞時，本文件優先於 AI 的通用語氣習慣。

若 AI 傾向使用現代心理學詞、抽象主題詞、通用奇幻詞或過度文藝句，而本文件禁止，則以本文件為準。

---

## 2.2 世界語言不等於漂亮語言

本文件的目標不是讓所有台詞變得華麗，而是讓台詞符合世界、階級、角色與場景。

允許：

- 粗糙
- 短促
- 錯誤
- 偏見
- 不完整
- 地方性
- 職業性
- 階級性
- 語言禁忌
- 非標準稱呼

不追求所有角色都講出工整、文學、成熟的句子。

---

## 2.3 世界觀資訊不得直接朗讀

角色不應自然地朗讀設定。

錯誤方向：

- 角色直接解釋歷史、制度、派系、魔法規則
- 角色使用百科式正式名詞
- 所有人都用同一套官方說法
- 對話像任務簡報或 wiki 條目

正確方向：

- 角色依自己的階級、職業、地域、信仰使用不同說法
- 重要概念有官方說法，也有俗稱、黑話、禁忌稱呼
- 角色可能說錯、避開、故意貶稱或不敢直呼
- 玩家透過多個局部說法拼出世界真相

# 3. 整體語言基調

本節定義全作品最基本的語言方向。

## 3.1 作品語言一句話定義

請用一句話描述本作品台詞整體語言基調：

> 例如：低魔奇幻世界中的現代白話台詞，帶有階級稱謂與宗教禁忌，但避免古風腔與通用日式 RPG 翻譯腔。

本作品語言一句話定義：

> 賽博 / 後啟示錄調的現代軍事承包白話，帶厭世吐槽、成本 / 合約 / 報表術語與黑市黑話，刻意避免史詩奇幻腔、日式 RPG 熱血宣言腔與古風腔；無宗教語、無神名。

## 3.2 語言年代感

本作品實際採用：

> **現代口語＋軍事任務語體**為主，混入**黑幫 / 街頭口語**（外圍、黑市、傭兵）。**不採用**古典白話、武俠白話、宮廷禮法、低魔奇幻、史詩奇幻中譯腔、日式 RPG 中譯腔。

| 項目 | 選擇 / 說明 |
|---|---|
| 現代口語 | 採用（主體；但去除網路時代用語） |
| 軍事 / 科幻任務語體 | 採用（官方語：作戰部、AI 客服、合約條款） |
| 黑幫 / 街頭口語 | 採用（民間語 / 階級語：外圍、黑市、狩獵隊、底層傭兵） |
| 史詩奇幻中譯腔 / 日式 RPG 中譯腔 | 禁止 |
| 宮廷禮法語體 / 古典白話 / 武俠白話 | 不適用（無古典 / 王室階層） |

---

## 3.3 口語程度

| 等級 | 說明 | 是否採用 |
|---|---|---|
| 高口語 | 接近日常對話，可破句、髒話、語助詞多 | 採用（前線、外圍、黑市） |
| 中口語 | 可自然對話，但避免太現代或網路語 | 採用（一般敘事預設） |
| 低口語 | 句子較完整，適合正式、歷史或儀式感世界 | 限用（官方文書 / AI 客服制式訊息） |
| 儀式語 | 用於宗教、法庭、王室、咒語等場合 | 不適用（無宗教 / 王室） |

本作品預設口語程度：

> 中口語為基準；前線 / 外圍 / 黑市場景升為高口語（可破句、髒話、省略主詞）；官方終端 / AI 客服 / 合約條款降為低口語制式語體（冷感、禮貌、無情、重流程）。

不同場合例外（對齊議題 10.7 三層：官方語 / 民間語 / 階級語）：

- **官方語（軍方 / 組織 / 合約）**：制式、禮貌、無情、流程導向；把見死不救包裝成條款（例：「依目前任務分類，該區域尚未達追加補給標準」）。
- **民間語（外圍居民 / 平民）**：粗糙、短促、實用，圍繞補給、危險、生計；無宗教感嘆，感嘆源多為世俗化粗話與抱怨。
- **階級語（前線老手對上層）**：表面服從、底層厭世吐槽；用黑色幽默消極抵抗（例：「黑翼最厲害的武器不是火炮，是把見死不救寫成合約條款」）。
- 軍人 / 傭兵：火藥、火炮、彈藥、座標、撤退時機等職業詞；情緒退回最熟悉的職業語言。
- 技術人員（莉娜 / 諾拉）：莉娜＝機械 / 火藥 / 火炮可靠性術語；諾拉＝資訊 / 能源 / 規則型 AI 術語（兩者衝突見 01_c）。
- 地下勢力 / 黑市：隱晦、交易感、對官方稱呼嘲諷。
- 宗教人士 / 怪物非人：不適用（無宗教；蟲群非語言性威脅）。

# 4. 本作品不該像什麼

本節用於排除錯誤文風。  
比起只寫「要自然」，更應明確寫「不要像什麼」。

## 4.1 禁止模仿的語感

本作品台詞不應像：

1.
2.
3.
4.
5.

可選參考類型：

- 現代職場簡報
- 網路雞湯文
- 通用奇幻小說翻譯腔
- 日式 RPG 熱血宣言腔
- 過度文青散文
- 古裝劇假古風
- 企劃書式任務說明
- 心理諮商式自我分析
- 所有人都情商很高的對話
- 角色替作者講主題的對話

# 5. 本作品應該像什麼

本節用於建立正向語言方向，但要避免過度抽象。

## 5.1 目標語感

本作品台詞應該具備以下特質：

1. 粗糙、短促、帶攻防（前線傭兵與外圍生活磨出來的語感）。
2. 厭世吐槽與黑色幽默（主角對高層、補給、津貼、合約的冷嘲）。
3. 冷感官僚腔（組織 / AI 客服把人命寫成流程與報表）。
4. 有軍事紀律與職業術語感（火炮、彈藥、座標、火控）。
5. 角色不習慣說真心；情緒透過行動、檢查裝備、責備、稱呼變化暴露。

對齊可選特質：克制、粗糙、短促、冷感、帶攻防、有階級感、有地方感、有軍事紀律、有黑市黑話、情緒透過行動與稱呼暴露、世界觀資訊從錯誤稱呼與俗稱中流出。**不採用**「有宗教壓力」（本作無宗教）。

## 11.2 職業語言使用規則

- 職業語言不應過量。

- 職業詞應在壓力、判斷、攻擊、安慰時自然流出。

- 角色越想展示權威，越可能使用正式術語。
- 角色越緊張，越可能退回最熟悉的職業語言。

# 23. 最終原則

世界語言規格的目標，是讓角色說出的每一句話都像是從這個世界的土壤裡長出來，而不是從通用 AI 劇本語料中抽出來。

本文件應幫助 AI 判斷：

- 這個詞在這個世界是否存在？
- 這個角色是否有資格使用這個詞？
- 這個階級是否會這樣稱呼對方？
- 這個陣營是否會使用官方說法？
- 這句話是否太像現代人？
- 這句話是否太像設定說明？
- 這句話是否太像通用奇幻或科幻台詞？

世界觀不是角色台詞的裝飾。  
世界觀應該限制角色、壓迫角色、改變角色說話的方式。
