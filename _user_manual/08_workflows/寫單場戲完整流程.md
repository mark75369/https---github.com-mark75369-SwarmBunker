# 情境 3：寫單場戲完整流程

> ⏳ **本文為骨架**，待 Phase D 完成後補真實對話 / 截圖。

# 1. 前提

Bible 已建好（情境 1 或 2 完成）。

# 2. 預估工時

**單場戲 30-60 分鐘**（從零到 LOCKED）。

# 3. 完整 8 步流程

## Step 1：F2 切到場景

前端 Scene Queue 找到 S-01-03（第一章第 3 場），點進 Scene Detail。

## Step 2：跑 /scene-task 產任務包

點「📋 複製 /scene-task 指令」→ 貼到 Claude Code → agent 自動裝載 context（C-* 聲線卡 / R-* 關係 / 05_d 揭露 / 05_e 伏筆 / 等）→ 產出任務包。

升 TASK_REVIEW（過 D.2.5 gate — user 拍板）。

## Step 3：跑 /dialogue-write 產 3 版

點「📋 複製 /dialogue-write 指令」→ 貼到 CC → agent 產 v01A / v01B / v01C 三版。

## Step 4：F3 三欄並排挑亮點

切回前端 F3 看三版並排：

```
v01A 克制版    │  v01B 衝突版   │  v01C 情緒版
…             │  …            │  …
```

框選你喜歡的句子（如 v01A 第 1 句 + v01C 第 2 句）。

## Step 5：跑 /dialogue-write --converge 收斂

點「📋 複製收斂指令」（含你挑的 picks）→ 貼到 CC → agent 整合產 v02。

升 DIALOGUE_CONVERGED（過 D.3.5 gate — user 拍板）。

## Step 6：跑 /qa 跑 8 份 QA

點「📋 複製 /qa 指令」→ 貼到 CC → agent 並行跑 8 份報告 → 產 9 個檔（8 QA + 0 個 09_e）。

## Step 7：F7 修補 + 升 FINAL

看 QA Combined Report：

- 全 PASS → 直接升 FINAL
- 有 WARN / FAIL → F7 直接點台詞改 → Save → 重跑 /qa

升 FINAL 前**填 09_e**（人類 final-gating 紀錄；可選簡寫）。

## Step 8：升 LOCKED

確定不再改 → 升 LOCKED。

---

# 4. 重要原則

1. **agent 給 3 版供選，你拍板** — 不要讓 agent 自己挑（會偏 AI 味）
2. **QA 8 份不能跳** — FAIL 一份就要改
3. **狀態升級都要 user 明示** — agent 不擅自升
4. **LOCKED 後想改要降級** — 走 09_e 紀錄流程
