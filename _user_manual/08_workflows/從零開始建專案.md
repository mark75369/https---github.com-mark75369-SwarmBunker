# 情境 1：從零開始建專案

> ⏳ **本文為骨架**，待 Phase B/C 完成後補真實對話 / 截圖。

# 1. 適合對象

純新作 / 完全從零開始的 user — 沒有任何既有手稿，跟 agent 對話從零建設定。

# 2. 預估工時

- Bible 建立（世界觀 / 角色 / 主線 / 細綱）：**2-3 天密集**
- 之後進入日常寫對白循環

# 3. Step-by-step

## Day 1：啟動 + 世界觀

1. `git clone` 工具 repo
2. 開 Claude Code，跑 `/init-project`，5 分鐘建專案結構
3. `python serve.py` 啟動前端
4. 跑 `/create-world` — 11 議題，2-3 小時
5. 跑 `/status` 確認進度

## Day 2：角色

6. 跑 `/create-character 主角A` — 9 議題，30-60 分鐘
7. 重複 step 6 × N 重要角色
8. 跑 `/create-relationship 主角A 反派B` — 6 議題，30 分鐘
9. 重複 step 8 × M 重要關係

## Day 3：主線 + 細綱

10. 跑 `/create-outline` — 7 議題，1-2 小時
11. 跑 `/create-detailed-outline` — 7 議題 + 場景索引，半天
